## 1. 랜딩 페이지 (/)

### 구성 요소

| 섹션 | 설명 |
|------|------|
| **Hero 섹션** | 메인 슬로건, CTA 버튼 (시작하기, 로그인) |
| **기능 소개** | 주요 기능 카드 (스터디 탐색, 실시간 협업 등) |
| **통계 섹션** | 사용자 수, 스터디 수 등 (계획) |
| **Footer** | 링크, 소셜, 저작권 |
