## 🎯 문서 활용 방법

### PPT 제작 시

1. **09_PPT-가이드.md**를 먼저 읽고 전체 슬라이드 구성 파악
2. 각 슬라이드에 해당하는 문서에서 내용 발췌
3. 스크린샷은 실제 앱을 실행하여 촬영
4. 데모 영상은 이미 제작되어 있으므로 PPT에 링크 삽입

### 권장 PPT 슬라이드 수

- **간략 버전**: 10-12장 (핵심만)
- **표준 버전**: 15-20장 (권장)
- **상세 버전**: 20-25장 (모든 내용)

| 항목 | 내용 |

## 🛠 기술 스택 요약

```
Frontend:  Next.js 16 + React 19 + Tailwind CSS 4 + TanStack Query 5
Backend:   Next.js API Routes + Prisma 6 + NextAuth.js 4 + Zod 4
Realtime:  Socket.IO 4 + WebRTC + Redis Adapter
Database:  PostgreSQL 15 + Redis 7
Infra:     Docker + Docker Compose
```
