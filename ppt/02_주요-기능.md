## 1. 스터디 관리

### 스터디 탐색 (Studies Explore)

| 기능 | 설명 |
|------|------|
| **카테고리별 검색** | 프로그래밍, 어학, 자격증, 취미, 독서, 재테크 등 |
| **키워드 검색** | 스터디명, 설명, 태그 기반 검색 |
| **필터링** | 모집 상태(모집중/마감), 공개/비공개 |
| **무한 스크롤** | 대량의 스터디 목록을 효율적으로 로딩 |
| **스터디 카드** | 이모지, 이름, 설명, 멤버 수, 모집 상태 표시 |

### 스터디 생성

| 기능 | 설명 |
|------|------|
| **기본 정보** | 스터디명, 이모지, 설명 |
| **카테고리 설정** | 메인 카테고리, 서브 카테고리 |
| **멤버 설정** | 최대 인원 (기본 20명), 공개/비공개 |
| **가입 설정** | 자동 승인 / 수동 승인 |
| **태그 설정** | 스터디를 설명하는 키워드 태그 |
| **초대 코드** | 자동 생성되는 고유 초대 코드 |

### 멤버 관리

| 역할 | 권한 |
|------|------|
| **👑 OWNER (방장)** | 모든 권한 - 스터디 삭제, 관리자 임명, 멤버 강퇴 등 |
| **⭐ ADMIN (관리자)** | 멤버 관리, 공지 작성, 일정/과제 관리 |
| **👤 MEMBER (멤버)** | 기본 참여 권한 - 채팅, 파일 공유, 일정 확인 |

### 가입 승인 시스템

- **수동 승인 모드**: 가입 신청 → 관리자 검토 → 승인/거부
- **자동 승인 모드**: 가입 신청 즉시 멤버로 등록
- **가입 동기/소개**: 신청 시 자기소개 및 가입 동기 작성

## 2. 실시간 협업

### 실시간 채팅

| 기능 | 설명 |
|------|------|
| **그룹 채팅** | 스터디별 전용 채팅방 |
| **실시간 메시지** | Socket.IO 기반 즉각적인 메시지 전송/수신 |
| **파일 첨부** | 채팅 중 이미지, 문서 파일 공유 |
| **메시지 읽음 표시** | 메시지별 읽음 상태 추적 |
| **타이핑 인디케이터** | 상대방 입력 중 표시 |
| **메시지 삭제** | 본인 메시지 삭제 기능 |

### 화상 통화 (WebRTC)

| 기능 | 설명 |
|------|------|
| **실시간 화상 통화** | WebRTC 기반 P2P 영상 통화 |
| **화면 공유** | 발표 자료, 코드 등 화면 공유 |
| **음소거/비디오 제어** | 마이크, 카메라 ON/OFF |
| **참여자 목록** | 현재 통화 참여자 표시 |
| **통화 시간** | 진행 중인 통화 시간 표시 |
| **설정** | 오디오/비디오 장치, 품질 설정 |

### 파일 공유

| 기능 | 설명 |
|------|------|
| **파일 업로드** | 학습 자료, 과제 제출 등 |
| **다운로드 카운트** | 파일별 다운로드 횟수 추적 |
| **파일 관리** | 업로더, 날짜, 크기 등 메타데이터 표시 |

## 3. 학습 관리

### 캘린더

| 기능 | 설명 |
|------|------|
| **일정 생성** | 스터디 정기 모임, 특별 이벤트 등록 |
| **일정 조회** | 월별/주별 일정 확인 |
| **색상 구분** | 일정 유형별 색상 지정 |
| **시간 설정** | 시작/종료 시간, 장소 설정 |
| **일정 알림** | 다가오는 일정 알림 |

### 개인 할일 관리 (Tasks)

| 기능 | 설명 |
|------|------|
| **할일 생성** | 제목, 설명, 마감일, 우선순위 |
| **상태 관리** | TODO → IN_PROGRESS → REVIEW → DONE |
| **우선순위** | LOW, MEDIUM, HIGH, URGENT |
| **스터디 연결** | 특정 스터디와 연결 가능 |
| **필터/정렬** | 완료 상태, 마감일, 우선순위별 정렬 |

### 스터디 공유 할일 (Study Tasks)

| 기능 | 설명 |
|------|------|
| **공유 할일** | 스터디 전체 멤버가 볼 수 있는 과제 |
| **담당자 지정** | 특정 멤버에게 할일 배정 |
| **진행률 추적** | 멤버별 과제 완료 상태 확인 |

### 공지사항

| 기능 | 설명 |
|------|------|
| **공지 작성** | 제목, 내용 (마크다운 지원) |
| **중요 공지** | 중요 표시로 강조 |
| **고정 공지** | 상단 고정 기능 |
| **조회수** | 공지별 조회수 추적 |
| **파일 첨부** | 공지에 파일 첨부 가능 |

## 5. 알림 시스템

### 알림 유형

| 알림 | 설명 |
|------|------|
| **JOIN_APPROVED** | 스터디 가입 승인 알림 |

| **KICK** | 강퇴 알림 |

### 알림 관리

- **전체 읽음 처리** 기능

- **실시간 알림** 수신 (Socket.IO)
