# CoUp PPT 제작 가이드라인

---

## 🚀 PPT 만드는 방법 (3가지)

### ⭐ 방법 1: Gamma.app (가장 쉬움, 추천)

1. **https://gamma.app** 접속 후 로그인
2. "Create new" → "Paste in text" 선택
3. 아래 **[Gamma용 프롬프트]** 전체를 복사해서 붙여넣기
4. AI가 자동으로 슬라이드 생성
5. 각 슬라이드에 스크린샷 직접 업로드
6. PPT 또는 PDF로 내보내기

### 방법 2: Canva

1. **https://canva.com** 접속
2. "프레젠테이션" → "16:9 와이드스크린" 선택
3. 아래 가이드라인을 참고해 수동으로 제작
4. 스크린샷 업로드 및 배치
5. PPT/PDF 내보내기

### 방법 3: PowerPoint 직접 제작

1. PowerPoint 또는 Google Slides 열기
2. 아래 가이드라인의 각 슬라이드 레이아웃 참고
3. 텍스트와 이미지 수동 배치

슬라이드 3: 프로젝트 소개
- CoUp = Co(operate) + Up = 함께 성장
- 핵심: 스터디 탐색부터 협업까지, 올인원 스터디 플랫폼
- 개발 기간: 약 6주 (2025.10.28 ~ 12.07)
- 인원: 1인 풀스택
- 주요 기술: Next.js 16, React 19

슬라이드 4: 문제 인식
현재 스터디 모집 방식: 카카오톡 오픈채팅방

4가지 문제점:
1. 비효율적 검색 - 필터링 불가
2. 관리 기능 부재 - 일정/출석 관리 X
3. 도구 분산 - 채팅/Zoom/드라이브 분리
4. 동기부여 어려움 - 학습 통계 X

슬라이드 5: 해결 방안
핵심 메시지: "모든 스터디 기능을 하나의 플랫폼에 통합"

문제 → 해결 매핑:
- 비효율적 검색 → 카테고리/키워드 검색 & 필터
- 관리 기능 부재 → 일정/출석/과제 통합 관리
- 도구 분산 → 채팅+화상+파일 All-in-One
- 동기부여 어려움 → 출석률/완료율 통계 시각화

---

설명:

- 키워드 검색: 스터디명, 설명, 태그 검색
- 모집 상태 필터: 모집중 / 모집마감

---

슬라이드 7: 기능 - 스터디 생성
레이아웃: 왼쪽에 스크린샷 영역, 오른쪽에 설명

설명:
- 기본 정보: 이름, 설명, 이모지
- 인원 설정: 최대 100명까지
- 가입 승인 방식: 자동 승인 / 수동 승인
- 공개 설정: 공개 / 비공개

---

슬라이드 8: 기능 - 멤버 관리 (RBAC)
레이아웃: 왼쪽에 스크린샷 영역, 오른쪽에 설명

역할 기반 접근 제어:
- OWNER (방장): 모든 권한, 스터디 삭제/위임
- ADMIN (관리자): 멤버/공지/일정 관리, 가입 승인
- MEMBER (멤버): 기본 참여 권한, 채팅/파일 이용

---

슬라이드 9: 기능 - 실시간 채팅

Socket.IO 기반 실시간 통신:
- 실시간 메시지 전송
- 파일 첨부: 이미지, 문서 공유

---

슬라이드 10: 기능 - 화상 통화
레이아웃: 왼쪽에 스크린샷 영역, 오른쪽에 설명

WebRTC 기반 P2P 연결:
- 서버 거치지 않고 직접 연결
- 다중 참여자: 여러 명 동시 통화
- 화면 공유: 발표 자료, 코드 공유
- 미디어 제어: 카메라/마이크 ON/OFF

---

슬라이드 11: 기능 - 대시보드
레이아웃: 왼쪽에 스크린샷 영역, 오른쪽에 설명

학습 현황 한눈에 확인:

슬라이드 12: 기능 - 학습 관리
