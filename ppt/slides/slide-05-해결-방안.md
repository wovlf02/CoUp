## 📝 들어가는 내용 (정확히)

**핵심 메시지** (상단 강조 박스)
- "모든 스터디 기능을 하나의 플랫폼에 통합" - 28pt

**문제 → 해결 매핑 표** (4행)

| 문제 | → | 해결 |
|------|---|------|
| 비효율적 검색 | → | 카테고리/키워드 검색 & 필터 |
| 관리 기능 부재 | → | 일정/출석/과제 통합 관리 |
| 도구 분산 | → | 채팅+화상+파일 All-in-One |
| 동기부여 어려움 | → | 출석률/완료율 통계 시각화 |

---

⏱️ **발표 시간**: 30-35초

💬 **스크립트**:
> "그래서 CoUp은 모든 스터디 기능을 하나로 통합합니다.
> 검색은 카테고리와 키워드로 쉽게,
> 일정, 출석, 과제를 한곳에서 관리하고,
> 채팅, 화상 통화, 파일 공유를 올인원으로 제공합니다.
> 학습 통계를 시각화해서 동기부여도 돕습니다."
