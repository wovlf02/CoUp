# PPT 슬라이드 설계

이 폴더는 CoUp 프로젝트 발표용 PPT 슬라이드 설계 문서입니다.

---

## 📄 통합 문서

### 👉 [ALL-SLIDES.md](./ALL-SLIDES.md)

**모든 슬라이드(17장)를 한 문서에 통합**한 파일입니다.
PPT 제작 시 이 파일만 참고하시면 됩니다.

---

## 📏 PPT 설계 기준

| 항목 | 기준 |
|------|------|
| 슬라이드 비율 | 16:9 |
| 제목 폰트 | 44pt Bold |
| 본문 폰트 | 18-24pt |
| 메인 색상 | #6366F1 (보라) |
| 총 슬라이드 | 17장 |
| 총 발표 시간 | 약 10분 (데모 포함) |

## 🖼️ 필요 스크린샷 (4장)

1. **대시보드** (`/dashboard`) - 통계 카드, 내 스터디
2. **스터디 탐색** (`/studies`) - 카테고리, 스터디 카드
3. **채팅** (`/my-studies/[id]/chat`) - 메시지, 온라인 멤버
4. **화상 통화** (`/my-studies/[id]/video-call`) - 비디오, 컨트롤
