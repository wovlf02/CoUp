## 🎯 Phase 3 범위

### 내 스터디 영역 개요

**내 스터디(My Studies)**는 사용자가 **참여하고 있는 스터디**의 **내부 활동 공간**으로, 다음 기능들을 포함합니다:

1. **내 스터디 목록** (`/my-studies`)
   - 전체, 참여중, 관리중, 대기중 탭
   - 역할 배지 (OWNER, ADMIN, MEMBER, PENDING)
   - 페이지네이션

2. **스터디 대시보드** (`/my-studies/[studyId]`)
   - 이번 주 활동 요약
   - 스터디 소개
   - 최근 공지/파일/일정/할일 위젯
   - 탭 네비게이션

3. **공지사항** (`/my-studies/[studyId]/notices`)
   - CRUD 작업 (ADMIN+)
   - 고정 공지 (최대 3개)
   - 검색/필터링

4. **할일 관리** (`/my-studies/[studyId]/tasks`)
   - 상태 관리 (TODO, IN_PROGRESS, REVIEW, DONE)
   - 칸반/리스트 뷰
   - 담당자 할당
   - 우선순위 및 마감일

5. **파일 관리** (`/my-studies/[studyId]/files`)
   - S3 업로드/다운로드
   - 용량 제한 (10MB)
   - 형식 검증
   - 드래그 앤 드롭

6. **캘린더** (`/my-studies/[studyId]/calendar`)
   - 월/주/일 뷰
   - 반복 일정
   - 알림 설정
   - 종일/시간 일정

7. **채팅** (`/my-studies/[studyId]/chat`)
   - 실시간 메시지 (Pusher)
   - 무한 스크롤
   - 파일 첨부 (선택적)

8. **위젯 시스템**
   - 활동 요약
   - 최근 공지/파일/일정/할일
   - 자동 갱신
