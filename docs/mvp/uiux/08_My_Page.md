# 08. 마이페이지 (My Page - Profile Management)

## 1. 목적
사용자가 자신의 개인 정보(프로필)를 조회하고 수정하며, 계정 관련 설정을 관리할 수 있도록 합니다.

## 2. 주요 요소
*   **상단 헤더**: CoUp 로고, 사용자 프로필 이미지/닉네임, 알림 아이콘, 검색 바 (전역 헤더)
*   **좌측 사이드바**: 주요 메뉴 (전역 사이드바)
*   **페이지 타이틀**: "마이페이지" 또는 "프로필 관리"
*   **프로필 이미지**: 현재 프로필 이미지 표시 및 변경 버튼
*   **닉네임**: 현재 닉네임 표시 및 수정 입력 필드
*   **이메일**: 사용자 이메일 (수정 불가, 읽기 전용)
*   **자기소개**: 텍스트 에어리어 (수정 가능)
*   **저장 버튼**: 변경된 프로필 정보를 저장합니다.
*   **계정 탈퇴 버튼**: 계정 탈퇴 기능을 제공합니다.

## 3. 레이아웃
*   **2단 레이아웃**: 좌측 전역 사이드바, 우측 마이페이지 콘텐츠 영역으로 구성됩니다.
*   **콘텐츠 영역**: 프로필 이미지, 닉네임, 이메일, 자기소개 등의 정보가 카드 형태로 그룹화되어 표시됩니다.
*   **반응형 디자인**: 모바일 환경에서는 전역 사이드바가 햄버거 메뉴로 대체되고, 프로필 정보는 세로로 깔끔하게 배치됩니다.

## 4. 상호작용
*   **프로필 이미지 변경**: 이미지 업로드 팝업/모달을 통해 새로운 이미지를 선택하고 업로드합니다.
*   **닉네임 수정**: 입력 필드에 새로운 닉네임을 입력하고, 저장 버튼 클릭 시 유효성 검사(중복 확인 등)를 수행합니다.
*   **저장 버튼 클릭**: 변경된 프로필 정보를 백엔드 API로 전송하고, 성공 시 성공 메시지를 표시합니다.
*   **계정 탈퇴 버튼 클릭**: 경고 메시지와 함께 확인 팝업/모달을 띄워 사용자의 최종 확인을 거친 후 계정 탈퇴를 진행합니다.

## 5. 디자인 고려사항
*   **시각적 피드백**: 프로필 이미지 변경 시 미리보기 제공, 닉네임 중복 시 즉각적인 오류 메시지 표시 등 사용자에게 명확한 피드백을 제공합니다.
*   **보안 및 개인정보**: 이메일과 같은 민감 정보는 수정 불가로 표시하고, 계정 탈퇴 시 개인정보 처리 방침에 대한 안내를 명확히 합니다.
*   **버튼 스타일**: "저장" 버튼은 Primary Color로 강조하고, "계정 탈퇴" 버튼은 경고 색상(예: 빨간색)을 사용하여 신중한 접근을 유도합니다.
*   **사용자 경험**: 프로필 수정 과정이 복잡하지 않도록 직관적인 UI를 제공합니다.

## 6. UI 요소 상세 (Detailed UI Elements)

### 6.1. 상단 헤더 (Global Header) 및 좌측 사이드바 (Left Sidebar)
*   메인 대시보드 페이지와 동일한 구성 및 스타일을 따릅니다.

### 6.2. 페이지 타이틀 (Page Title)
*   **텍스트**: "마이페이지" 또는 "프로필 관리".
*   **스타일**: H2 태그, Noto Sans KR, 폰트 크기 28px, Neutral Dark Color, 좌측 정렬.

### 6.3. 프로필 카드 (Profile Card)
*   **구성**: 프로필 이미지, 닉네임, 이메일, 자기소개 입력 필드, 저장 버튼, 계정 탈퇴 버튼.
*   **레이아웃**: 단일 컬럼, 각 정보 필드 사이에 20px 간격.
*   **스타일**: 배경 흰색, 둥근 모서리 (8px), 가벼운 그림자 (shadow-sm), 패딩 30px.

### 6.4. 프로필 이미지 영역 (Profile Image Area)
*   **구성**: 현재 프로필 이미지 (원형), "변경" 버튼 (Secondary Outline Button).
*   **이미지**: 120x120px 원형, 중앙 정렬.
*   **변경 버튼**: 이미지 아래 중앙에 배치.

### 6.5. 닉네임 입력 필드 (Nickname Input Field)
*   **구성**: 라벨 ("닉네임"), 입력 필드, 유효성 검사 메시지 영역.
*   **스타일**: Input Field는 일반 텍스트 입력 필드와 동일한 스타일.
*   **유효성 검사**: 중복 확인, 길이 제한 등 실시간 피드백 제공.

### 6.6. 이메일 표시 (Email Display)
*   **구성**: 라벨 ("이메일"), 이메일 주소 텍스트.
*   **스타일**: 읽기 전용, Neutral Gray Color, 폰트 16px.

### 6.7. 자기소개 텍스트 에어리어 (Bio Textarea)
*   **구성**: 라벨 ("자기소개"), 텍스트 에어리어.
*   **스타일**: Textarea는 일반 텍스트 에어리어와 동일한 스타일. 최소 높이 100px.

### 6.8. 저장 버튼 (Save Button)
*   **구성**: "저장" 텍스트.
*   **스타일**: Primary Button, 높이 48px, 너비 120px.
*   **위치**: 프로필 카드 하단 우측.

### 6.9. 계정 탈퇴 버튼 (Delete Account Button)
*   **구성**: "계정 탈퇴" 텍스트.
*   **스타일**: Secondary Outline Button, 경고 색상 (Accent Color Red 계열), 높이 48px, 너비 120px.
*   **위치**: 프로필 카드 하단 좌측 또는 별도 섹션.
*   **모달 (Modal)**: 클릭 시 경고 메시지 ("정말로 계정을 탈퇴하시겠습니까? 모든 데이터가 삭제됩니다."), "확인" (경고 버튼), "취소" 버튼 포함.
