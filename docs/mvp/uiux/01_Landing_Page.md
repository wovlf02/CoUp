# 01. 랜딩 페이지 (Landing Page)

## 1. 목적
로그인하지 않은 사용자에게 CoUp 서비스의 핵심 가치와 기능을 소개하고, 회원가입/로그인으로 유도합니다.

## 2. 주요 요소
*   **네비게이션 바**: 로고, 서비스 소개, 로그인/회원가입 버튼
*   **히어로 섹션**: 서비스 슬로건, 핵심 이미지/영상, CTA(Call To Action) 버튼
*   **서비스 소개 섹션**: CoUp의 주요 기능(스터디 탐색, 그룹 관리, 실시간 소통 등)을 시각적으로 설명
*   **사용자 후기/사례 섹션**: CoUp을 통해 성공한 사용자들의 이야기
*   **푸터**: 회사 정보, 약관, 개인정보처리방침 등

## 3. 레이아웃
*   **상단 고정 네비게이션 바**: 스크롤 시에도 항상 상단에 고정되어 주요 메뉴에 접근성을 높입니다.
*   **중앙 정렬 콘텐츠**: 대부분의 콘텐츠는 중앙에 정렬되어 시각적인 안정감을 제공합니다.
*   **반응형 디자인**: 모바일 환경에서는 네비게이션 바가 햄버거 메뉴로 전환되고, 콘텐츠 블록이 세로로 재배치됩니다.

## 4. 상호작용
*   **CTA 버튼**: "지금 시작하기", "회원가입" 등의 버튼 클릭 시 회원가입/로그인 페이지로 이동합니다.
*   **스크롤 애니메이션**: 스크롤 시 각 섹션이 부드럽게 나타나는 애니메이션 효과를 적용하여 시각적인 흥미를 유발합니다.

## 5. 디자인 고려사항
*   **슬로건**: "함께, 더 높이. 당신의 성장을 위한 스터디 허브."를 강조하여 배치합니다.
*   **이미지/영상**: 서비스의 핵심 가치를 직관적으로 전달할 수 있는 고품질의 이미지나 짧은 영상을 활용합니다. (예: 함께 공부하는 모습, 목표를 달성하고 기뻐하는 모습)
*   **색상**: Primary Color와 Secondary Color를 적절히 사용하여 활기차고 신뢰감 있는 분위기를 연출합니다.
*   **타이포그래피**: 슬로건과 주요 헤드라인에는 큰 폰트와 강조 색상을 사용하여 시선을 집중시킵니다.

## 6. UI 요소 상세 (Detailed UI Elements)

### 6.1. 네비게이션 바 (Navigation Bar)
*   **구성**: 좌측에 CoUp 로고 (SVG), 중앙에 "서비스 소개", "기능", "가격" 등 메뉴 (텍스트 링크), 우측에 "로그인" (Secondary Button), "회원가입" (Primary Button)
*   **높이**: 64px (데스크톱), 56px (모바일)
*   **배경**: 스크롤 시 배경색 (#FFFFFF) 및 하단 그림자 (shadow-sm) 적용
*   **모바일**: 햄버거 아이콘 (Material Icons) 클릭 시 전체 화면 메뉴 (Full-screen overlay menu) 토글

### 6.2. 히어로 섹션 (Hero Section)
*   **구성**:
    *   **슬로건**: H1 태그, Noto Sans KR, 폰트 크기 48px (데스크톱), 32px (모바일), Primary Color 또는 Neutral Dark Color
    *   **서브 슬로건**: P 태그, Noto Sans KR, 폰트 크기 20px (데스크톱), 16px (모바일), Neutral Gray Color
    *   **CTA 버튼**: "지금 시작하기" (Primary Button, 폰트 크기 18px), "자세히 알아보기" (Secondary Outline Button, 폰트 크기 18px)
    *   **핵심 이미지/영상**: 섹션의 60% 너비 차지 (데스크톱), 100% 너비 (모바일), 서비스의 가치를 보여주는 일러스트 또는 짧은 비디오
*   **정렬**: 텍스트와 버튼은 중앙 정렬, 이미지/영상은 우측 또는 하단에 배치 (반응형)
*   **패딩**: 상하 120px (데스크톱), 80px (모바일)

### 6.3. 서비스 소개 섹션 (Feature Section)
*   **구성**: 3~4개의 기능 카드 (Feature Card) 그리드 형태로 배치
*   **기능 카드**:
    *   **아이콘**: Material Icons, Primary Color
    *   **제목**: H3 태그, Noto Sans KR, 폰트 크기 24px
    *   **설명**: P 태그, Noto Sans KR, 폰트 크기 16px
*   **레이아웃**: 3열 그리드 (데스크톱), 1열 또는 2열 그리드 (모바일)
*   **패딩**: 상하 80px

### 6.4. 사용자 후기/사례 섹션 (Testimonial Section)
*   **구성**: 2~3개의 사용자 후기 카드 (Testimonial Card) 슬라이더 또는 그리드 형태로 배치
*   **후기 카드**:
    *   **사용자 프로필 이미지**: 원형, 48x48px
    *   **사용자 닉네임**: 텍스트
    *   **후기 내용**: 인용구 형태, P 태그
*   **패딩**: 상하 80px
*   **배경**: Neutral Light Gray (#F5F5F5)

### 6.5. 푸터 (Footer)
*   **구성**: 회사 로고, 회사명, 주소, 연락처, 개인정보처리방침, 이용약관, 소셜 미디어 링크 (아이콘)
*   **레이아웃**: 텍스트는 좌측 정렬, 소셜 미디어 아이콘은 우측 정렬 (데스크톱), 모두 중앙 정렬 (모바일)
*   **배경**: Neutral Dark Color (#424242)
*   **텍스트 색상**: Neutral Light Gray (#E0E0E0)
