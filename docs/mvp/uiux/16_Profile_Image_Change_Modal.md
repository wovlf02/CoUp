# 16. 프로필 이미지 변경 모달 (Profile Image Change Modal)

## 1. 목적
사용자가 자신의 프로필 이미지를 업로드하거나 기존 이미지를 삭제하여 변경할 수 있도록 합니다.

## 2. 주요 요소
*   **모달 헤더**: "프로필 이미지 변경" 타이틀, 닫기 버튼 (X 아이콘)
*   **현재 프로필 이미지 미리보기**: 현재 설정된 프로필 이미지를 보여줍니다.
*   **이미지 업로드 영역**: 드래그 앤 드롭 또는 파일 선택 버튼을 통해 새로운 이미지를 업로드합니다.
*   **이미지 삭제 버튼**: 현재 프로필 이미지를 기본 이미지로 되돌립니다.
*   **저장 버튼**: 변경된 프로필 이미지를 저장하고 모달을 닫습니다.
*   **취소 버튼**: 변경사항을 폐기하고 모달을 닫습니다.

## 3. 레이아웃
*   **중앙 정렬 모달**: 화면 중앙에 오버레이와 함께 모달이 표시됩니다.
*   **콘텐츠 레이아웃**: 미리보기, 업로드 영역, 삭제 버튼, 저장/취소 버튼이 세로로 배치됩니다.
*   **버튼 그룹**: 하단 우측에 저장 버튼과 취소 버튼이 배치됩니다.
*   **반응형 디자인**: 모바일 환경에서는 모달이 전체 화면을 차지하거나, 내용이 스크롤 가능하도록 조정됩니다.

## 4. 상호작용
*   **닫기 버튼/오버레이 클릭**: 모달이 닫힙니다.
*   **이미지 업로드**: 드래그 앤 드롭 또는 파일 선택 버튼 클릭 시 파일 탐색기가 열리고, 이미지 선택 후 미리보기가 업데이트됩니다.
*   **이미지 삭제 버튼 클릭**: 미리보기가 기본 이미지로 변경됩니다.
*   **저장 버튼 클릭**: 백엔드 API를 호출하여 프로필 이미지를 업데이트하고, 성공 시 모달을 닫고 마이페이지를 새로고침합니다.
*   **취소 버튼 클릭**: 모달을 닫고 변경사항을 폐기합니다.

## 5. 디자인 고려사항
*   **실시간 미리보기**: 새로운 이미지를 업로드하거나 삭제할 때 실시간으로 미리보기를 업데이트하여 사용자에게 즉각적인 피드백을 제공합니다.
*   **파일 형식/크기 제한**: 업로드 가능한 이미지 파일 형식(JPG, PNG 등) 및 크기 제한을 명확히 안내합니다.
*   **버튼 스타일**: "저장" 버튼은 Primary Color로 강조하고, "삭제" 버튼은 경고 색상(Accent Color Red 계열)을 사용합니다.
*   **피드백**: 업로드/저장 중에는 로딩 스피너를 표시하고, 성공/실패 시 토스트 메시지 등의 알림을 제공합니다.

## 6. UI 요소 상세 (Detailed UI Elements)

### 6.1. 모달 오버레이 (Modal Overlay)
*   **구성**: 화면 전체를 덮는 반투명 배경.
*   **스타일**: 배경색 Neutral Dark Color (opacity 0.5), 클릭 시 모달 닫힘.

### 6.2. 모달 컨테이너 (Modal Container)
*   **구성**: 모달 헤더, 미리보기, 업로드 영역, 삭제 버튼, 버튼 그룹.
*   **크기**: 데스크톱: 너비 450px, 높이 자동. 모바일: 너비 90%, 높이 자동.
*   **배경**: 흰색 (#FFFFFF).
*   **스타일**: 둥근 모서리 (8px), 그림자 (shadow-xl), 중앙 정렬.
*   **패딩**: 내부 패딩 30px.

### 6.3. 모달 헤더 (Modal Header)
*   **구성**: 타이틀 (H3), 닫기 버튼 (Material Icons - close).
*   **스타일**: 타이틀 폰트 20px, 굵은 글씨. 닫기 버튼은 우측 상단에 배치.

### 6.4. 현재 프로필 이미지 미리보기 (Current Profile Image Preview)
*   **구성**: 원형 이미지 (100x100px), 중앙 정렬.
*   **스타일**: 테두리 (Neutral Light Gray), 배경 Neutral Light Gray.

### 6.5. 이미지 업로드 영역 (Image Upload Area)
*   **구성**: 드래그 앤 드롭 영역, "파일 선택" 버튼, 가이드 텍스트 ("여기에 이미지를 드래그하거나 클릭하세요.").
*   **스타일**: 점선 테두리 (Neutral Light Gray), 둥근 모서리 (8px), 배경 Neutral Light Gray Light, 높이 120px.
*   **상태**: 파일 드래그 시 테두리 색상 Primary Color로 변경.

### 6.6. 이미지 삭제 버튼 (Delete Image Button)
*   **구성**: "이미지 삭제" 텍스트.
*   **스타일**: Secondary Outline Button, 경고 색상 (Accent Color Red 계열), 높이 40px.
*   **위치**: 업로드 영역 아래 중앙.
*   **상태**: 현재 프로필 이미지가 기본 이미지가 아닐 때만 활성화.

### 6.7. 버튼 그룹 (Button Group)
*   **구성**: "취소" 버튼 (Secondary Outline Button), "저장" 버튼 (Primary Button).
*   **정렬**: 우측 하단 정렬, 버튼 사이에 16px 간격.
*   **크기**: 높이 48px.
