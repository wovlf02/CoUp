# 02. 로그인/회원가입 페이지 (Auth Page)

## 1. 목적
사용자가 CoUp 서비스에 안전하고 간편하게 로그인하거나 새로운 계정을 생성할 수 있도록 합니다.

## 2. 주요 요소
*   **페이지 타이틀**: "로그인" 또는 "회원가입"
*   **소셜 로그인 버튼**: Google, GitHub (MVP)
*   **구분선**: "또는" 문구와 함께 소셜 로그인과 일반 로그인/회원가입 폼을 시각적으로 구분
*   **일반 로그인/회원가입 폼**: 이메일, 비밀번호 입력 필드 (MVP에서는 소셜 로그인만 지원하므로 이 부분은 Post-MVP)
*   **비밀번호 찾기/계정 찾기 링크** (Post-MVP)
*   **회원가입/로그인 전환 링크**: "계정이 없으신가요? 회원가입" 또는 "이미 계정이 있으신가요? 로그인"

## 3. 레이아웃
*   **중앙 정렬된 카드 형태**: 페이지 중앙에 깔끔한 카드 형태로 로그인/회원가입 폼을 배치하여 사용자의 집중도를 높입니다.
*   **간결한 디자인**: 불필요한 요소는 최소화하고, 로그인/회원가입 기능에만 집중합니다.
*   **반응형 디자인**: 모바일 환경에서도 폼이 잘 보이도록 크기를 조절하고, 버튼 등의 요소가 터치하기 쉽게 배치됩니다.

## 4. 상호작용
*   **소셜 로그인 버튼 클릭**: 각 소셜 서비스의 인증 페이지로 리디렉션됩니다.
*   **폼 제출**: 유효성 검사를 통과한 후 백엔드 API를 통해 인증을 시도합니다.
*   **오류 메시지**: 유효성 검사 실패 또는 인증 오류 시, 사용자에게 명확하고 친절한 오류 메시지를 표시합니다.
*   **전환 링크 클릭**: 로그인 페이지와 회원가입 페이지 간에 부드럽게 전환됩니다.

## 5. 디자인 고려사항
*   **브랜딩 일관성**: CoUp의 로고와 색상 팔레트를 사용하여 랜딩 페이지와의 일관성을 유지합니다.
*   **소셜 로그인 아이콘**: 각 소셜 서비스의 공식 아이콘과 색상을 사용하여 사용자가 쉽게 인지할 수 있도록 합니다.
*   **쉬운 접근성**: 입력 필드와 버튼의 크기를 적절하게 조절하여 모바일 환경에서도 사용하기 편리하도록 합니다.
*   **보안 강조**: 비밀번호 입력 시 마스킹 처리, 보안 관련 안내 문구 등을 통해 사용자의 신뢰를 얻습니다.

## 6. UI 요소 상세 (Detailed UI Elements)

### 6.1. 페이지 컨테이너 (Page Container)
*   **구성**: 전체 화면을 덮는 컨테이너.
*   **배경**: Neutral Light Gray (#F5F5F5) 또는 부드러운 그라데이션.
*   **정렬**: 중앙에 인증 카드 배치.

### 6.2. 인증 카드 (Auth Card)
*   **구성**: CoUp 로고, 페이지 타이틀, 소셜 로그인 버튼, 구분선, 전환 링크.
*   **크기**: 데스크톱: 최대 너비 400px, 높이 자동. 모바일: 너비 90%, 높이 자동.
*   **배경**: 흰색 (#FFFFFF).
*   **스타일**: 둥근 모서리 (border-radius: 8px), 그림자 (shadow-lg) 적용.
*   **패딩**: 내부 패딩 40px.

### 6.3. CoUp 로고
*   **위치**: 카드 상단 중앙.
*   **크기**: 높이 40px.

### 6.4. 페이지 타이틀 (Page Title)
*   **텍스트**: "로그인" 또는 "회원가입".
*   **스타일**: H2 태그, Noto Sans KR, 폰트 크기 28px, Neutral Dark Color, 중앙 정렬, 로고 아래 20px 간격.

### 6.5. 소셜 로그인 버튼 (Social Login Buttons)
*   **구성**: Google, GitHub 아이콘과 텍스트 ("Google로 로그인", "GitHub으로 로그인").
*   **스타일**:
    *   **Google**: 배경 흰색, 테두리 Neutral Light Gray, 텍스트 Neutral Dark Color. Google 로고 (SVG).
    *   **GitHub**: 배경 #24292E, 텍스트 흰색. GitHub 로고 (SVG).
*   **크기**: 너비 100%, 높이 48px.
*   **간격**: 각 버튼 사이에 12px 간격.
*   **상태**:
    *   **Hover**: 배경색 미묘하게 변화, 그림자 효과.
    *   **Active**: 배경색 더 진하게 변화.

### 6.6. 구분선 (Divider)
*   **구성**: 양쪽에 선이 있고 중앙에 "또는" 텍스트.
*   **스타일**: 텍스트 Neutral Gray Color, 선 Neutral Light Gray Color, 폰트 크기 14px.
*   **간격**: 소셜 로그인 버튼과 전환 링크 사이에 24px 간격.

### 6.7. 전환 링크 (Toggle Link)
*   **텍스트**: "계정이 없으신가요? 회원가입" 또는 "이미 계정이 있으신가요? 로그인".
*   **스타일**: P 태그, 폰트 크기 14px, Neutral Gray Color, 중앙 정렬. "회원가입" 또는 "로그인" 부분은 Secondary Color로 강조된 링크.
*   **상태**:
    *   **Hover**: 밑줄 효과.
