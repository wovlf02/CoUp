# UI/UX 설계 문서

이 폴더에는 CoUp 프로젝트의 UI/UX 설계 문서들이 포함되어 있습니다.

## 📑 문서 목록

### 1. [settings-page.md](./settings-page.md)
- **사용자 설정 페이지 설계**
- 프로필 편집
- 비밀번호 변경
- 알림 설정
- 테마 설정
- UI/UX 상세 명세

### 2. [system-settings.md](./system-settings.md)
- **시스템 설정 페이지 설계** (관리자용)
- 서비스 설정
- 제한 설정
- 관리자 계정 관리
- 백업 및 로그

## 🎯 설계 문서 작성 규칙

### 포함 내용
- 화면 구조 및 레이아웃
- 컴포넌트 구성
- 인터랙션 및 상태 관리
- API 연동 방식
- 권한 및 접근 제어

### 형식
```markdown
# 페이지명

## 개요
- 목적
- 접근 권한

## 화면 구조
- 레이아웃 다이어그램
- 섹션 설명

## 주요 기능
- 기능 목록
- 인터랙션

## API 연동
- 사용되는 API 목록
- 데이터 흐름

## 컴포넌트
- 주요 컴포넌트 목록
- props 및 state

## 반응형 디자인
- 데스크톱/태블릿/모바일 대응
```

## 📝 업데이트 규칙

- 실제 구현과 일치하도록 유지
- 주요 UI 변경 시 문서 업데이트
- 스크린샷 또는 다이어그램 포함
