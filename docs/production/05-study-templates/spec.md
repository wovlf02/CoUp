# Production 기능 명세: 05. 스터디 템플릿

## 1. 개요

- 사용자가 스터디 그룹을 생성할 때, 스터디 목적에 맞는 템플릿을 선택하여 그룹을 더 쉽게 설정하고, 유형에 맞는 전용 기능들을 사용할 수 있도록 한다.
- 이를 통해 다양한 분야의 사용자들이 자신의 학습 목표에 최적화된 환경을 손쉽게 구축할 수 있다.

## 2. 사용자 스토리 (User Stories)

### 2.1. 템플릿 선택 및 그룹 생성

- **US-1.1**: 사용자가 '스터디 생성' 버튼을 누르면, 가장 먼저 스터디 유형 템플릿을 선택하는 화면이 표시된다.
- **US-1.2**: 사용자는 다음과 같은 템플릿 중 하나를 선택할 수 있다.
  - **일반 스터디**: 특정 유형에 국한되지 않는 자유로운 스터디 (기본값)
  - **시험/자격증 준비**: 시험 준비에 특화된 템플릿
  - **어학 스터디**: 언어 학습에 특화된 템플릿
  - **독서 토론**: 책을 읽고 토론하는 것에 특화된 템플릿
  - **코딩 스터디**: 개발자들을 위한 템플릿
- **US-1.3**: 템플릿을 선택하면, 해당 템플릿에 맞는 추천 그룹 규칙, 기본 메뉴 구성, 전용 기능들이 제안되거나 자동으로 설정된다.

### 2.2. 템플릿별 전용 기능

- **US-2.1 (시험/자격증 준비 템플릿)**
  - 그룹 생성 시 '목표 시험'과 'D-Day'를 설정할 수 있다.
  - 그룹 메인 페이지에 D-Day 카운터가 표시된다.
  - '모의고사/퀴즈' 기능(플래시카드)이 기본으로 활성화된다.
  - '학습 계획' 탭에서 주차별 학습 목표를 설정하고 진도율을 추적하는 기능이 제공된다.

- **US-2.2 (어학 스터디 템플릿)**
  - '오늘의 단어/문장'을 등록하고 공유하는 위젯이 제공된다.
  - '플래시카드' 기능이 기본으로 활성화된다.
  - 1:1 또는 그룹으로 '회화 연습'을 예약하고 진행할 수 있는 기능이 제공된다.

- **US-2.3 (독서 토론 템플릿)**
  - 그룹 생성 시 '이번 주/달의 책'을 ISBN 검색 등으로 쉽게 등록할 수 있다.
  - '토론 발제' 탭에서 멤버들이 토론하고 싶은 주제를 미리 올리고 투표할 수 있다.
  - '인상 깊은 구절'을 공유하고 의견을 나누는 공간이 제공된다.

- **US-2.4 (코딩 스터디 템플릿)**
  - 그룹 생성 시 관련 기술 스택(e.g., `React`, `Spring Boot`)을 태그로 등록할 수 있다.
  - 칸반 보드에 '코드 리뷰 요청' 컬럼이 기본으로 생성된다.
  - 그룹 멤버의 GitHub, 백준 등 계정을 연동하여 활동을 공유하는 기능이 강화된다.

## 3. 기술 아키텍처

- **템플릿 메타데이터**: 각 템플릿의 이름, 설명, 기본 설정(메뉴, 규칙), 활성화되는 전용 기능 목록 등을 JSON 형태로 관리한다.
- **동적 메뉴/컴포넌트 렌더링**: 프론트엔드는 그룹의 템플릿 유형에 따라 표시되는 메뉴나 컴포넌트를 동적으로 렌더링한다.
- **전략 패턴 (Strategy Pattern)**: Next.js API Routes에서 템플릿별로 달라지는 로직(예: 그룹 생성 시 초기 설정)을 처리할 때, 전략 패턴을 사용하여 각 템플릿의 로직을 캡슐화하고 확장성 있는 구조를 만듭니다.
