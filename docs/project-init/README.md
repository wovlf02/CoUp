# 프로젝트 초기 설정 문서

이 폴더에는 CoUp 프로젝트의 전반적인 내용과 초기 설정에 관한 문서들이 포함되어 있습니다.

## 📑 문서 목록

### 1. [overview.md](./overview.md)
- **프로젝트 종합 가이드**
- 프로젝트 개요, 비전, 핵심 가치
- 기술 스택 및 시스템 아키텍처
- MVP 목표 및 개발 로드맵
- 전체 프로젝트 구조 이해를 위한 핵심 문서

### 2. [file-structure.md](./file-structure.md)
- **파일 및 폴더 구조**
- Next.js 프로젝트 디렉토리 구조
- 파일 명명 규칙
- 컴포넌트 구조 및 배치

### 3. [database.md](./database.md)
- **데이터베이스 설계**
- Prisma 스키마 개요
- 주요 모델 설명
- 관계(Relation) 정의

### 4. [database-sql.md](./database-sql.md)
- **SQL 스키마**
- 실제 데이터베이스 테이블 구조
- SQL 마이그레이션 스크립트
- 인덱스 및 제약 조건

### 5. [api.md](./api.md)
- **API 명세서**
- RESTful API 엔드포인트
- 요청/응답 형식
- 인증 및 권한 관리

## 🎯 사용 방법

### 신규 개발자 온보딩
1. **overview.md** 먼저 읽기 → 프로젝트 전체 이해
2. **file-structure.md** 확인 → 코드 구조 파악
3. **database.md** & **database-sql.md** 학습 → 데이터 모델 이해
4. **api.md** 참조 → API 개발 및 사용

### 기존 개발자
- 특정 기능 개발 시 해당 문서 참조
- 아키텍처 변경 시 문서 업데이트 필수
