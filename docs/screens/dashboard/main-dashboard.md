## 🎨 섹션별 상세 설계

### 1. Header (Fixed, 64px)
```
┌──────────────────────────────────────────────────────────────┐
│ [🔍 스터디 검색...]              [🔔 3] [👤 김민준 ▼]        │
└──────────────────────────────────────────────────────────────┘
```

**좌측**:
- 검색바 (400px)
  - placeholder: "스터디 검색..."
  - 포커스 시 드롭다운 (최근 검색, 추천 스터디)

**우측**:
- 알림 아이콘 + 배지 (읽지 않은 알림 수)
  - 클릭 시 알림 드롭다운 (최근 5개)
- 프로필 드롭다운
  - 프로필 이미지 + 이름
  - 드롭다운: 마이페이지, 설정, 로그아웃

### 3. 통계 카드 (4개 그리드)
```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│  참여 스터디 │  │  새 공지    │  │  할 일      │  │  다가올 일정 │
│     4       │  │     3       │  │     5       │  │     2       │
└─────────────┘  └─────────────┘  └─────────────┘  └─────────────┘
```

**각 카드**:
- 크기: 1/4 너비 (4컬럼 그리드)
- 배경: white
- 테두리: 1px solid gray-200
- 둥근 모서리: 12px
- 패딩: 24px
- 그림자: shadow-sm

**내용**:
1. 아이콘 (32px, primary-500)
2. 제목 (text-sm, gray-600)
3. 숫자 (text-3xl, Bold, gray-900)

**카드 종류**:
1. 참여 스터디 (📚 아이콘)
2. 새 공지 (📢 아이콘)
3. 할 일 (✅ 아이콘, 미완료 개수)
4. 다가올 일정 (📅 아이콘, 7일 이내)

### 4. 내 스터디 섹션
```
내 스터디                                           [전체 보기 →]

┌──────────────────────┐  ┌──────────────────────┐
│ 📚                   │  │ 💼                   │
│ 코딩테스트 스터디      │  │ 취업 준비 스터디      │
│                      │  │                      │
│ 12명 참여 · OWNER    │  │ 8명 참여 · MEMBER    │
│ 마지막 활동: 1시간 전  │  │ 마지막 활동: 3시간 전 │
│                      │  │                      │
│ [채팅] [공지] [파일]  │  │ [채팅] [공지] [파일] │
└──────────────────────┘  └──────────────────────┘
```

**제목 바**:
- "내 스터디" (text-xl, Bold, gray-900)
- "[전체 보기 →]" 링크 (우측 정렬) → `/studies?filter=my`

**스터디 카드**:
- 크기: 1/2 너비 (2컬럼), 1/3 (3컬럼 Desktop)
- 배경: white
- 테두리: 1px solid gray-200
- 둥근 모서리: 12px
- 패딩: 20px
- Hover: shadow-md, 테두리 primary-200

**카드 내용**:
1. 카테고리 이모지 (상단, 32px)
2. 스터디명 (text-lg, Bold, gray-900)
3. 멤버 수 + 역할 (text-sm, gray-600)
4. 마지막 활동 시간 (text-xs, gray-500)
5. 빠른 액션 버튼 (하단)
   - [채팅] [공지] [파일]
   - 아이콘 + 텍스트, text-sm

**클릭** → `/studies/[studyId]`

### 5. 최근 활동 섹션
```
최근 활동                                           [전체 보기 →]

• [공지] 코딩테스트 스터디 - 이번 주 일정 공지               2시간 전
• [할일] 취업 준비 스터디 - 자소서 1차 작성 완료             3시간 전
• [파일] 영어 스터디 - 단어장.pdf 업로드됨                   5시간 전
• [채팅] 코딩테스트 스터디 - 김철수: 오늘 저녁 회의 참석 가능...  6시간 전
• [일정] 취업 준비 스터디 - 모의면접 (내일 오후 2시)         1일 전
```

**제목 바**:
- "최근 활동" (text-xl, Bold, gray-900)
- "[전체 보기 →]" → `/notifications`

**활동 아이템**:
- 배경: gray-50 (Hover: gray-100)
- 패딩: 16px
- 둥근 모서리: 8px
- 간격: 8px

**아이템 내용**:
1. 타입 배지 (좌측)
   - [공지], [할일], [파일], [채팅], [일정]
   - 배경: primary-100, 텍스트: primary-700
   - 패딩: 4px 8px, 둥근 모서리: 4px
2. 스터디명 (text-sm, gray-900)
3. 활동 내용 (text-sm, gray-700)
4. 시간 (우측, text-xs, gray-500)

**클릭** → 해당 활동 상세 페이지
