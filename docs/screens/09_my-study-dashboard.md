# 07. 스터디 상세 - 개요 (Study Detail - Overview)

> **화면 ID**: `STUDY-01`  
> **라우트**: `/studies/[studyId]`  
> **레이아웃**: 좌측 네비게이션(15%) + 우측 콘텐츠(85%) → **2컬럼 레이아웃 개선**  
> **렌더링**: Hybrid (SSR → CSR)

---

## 📐 화면 구조 개요 (개선안)

**문제점**: 우측 콘텐츠 영역(85%)에서 중앙 정렬로 인해 좌우 여백이 많이 낭비됨

**개선안**: 2컬럼 레이아웃 적용 - **메인 콘텐츠(1100px) + 우측 고정 위젯(280px)**

### ⭐ 우측 위젯 전역 표시 전략 (FHD 최적화)

**핵심 원칙**: 모든 탭에서 우측 위젯을 표시하여 공간 활용 극대화 및 UX 개선

| 탭 | 레이아웃 | 우측 위젯 | 설명 |
|---|---|---|---|
| **개요** | 2컬럼 | ✅ 전체 위젯 | 스터디 소개 + 전체 위젯 |
| **채팅** | 2컬럼 | ✅ 온라인 멤버 강조 | 채팅 + 실시간 멤버 현황 |
| **공지** | 2컬럼 | ✅ 전체 위젯 | 공지 목록 + 전체 위젯 |
| **파일** | 2컬럼 | ✅ 전체 위젯 | 파일 목록 + 전체 위젯 |
| **캘린더** | 2컬럼 | ✅ 일정 중심 위젯 | 캘린더 + 일정/할일 위젯 |
| **할일** | 2컬럼 | ✅ 할일 중심 위젯 | 할일 목록 + 진행률 위젯 |
| **화상 스터디** | 전체 화면 | ❌ 별도 | 전체 화면 모드 |
| **설정** | 2컬럼 | ✅ 전체 위젯 | 설정 폼 + 전체 위젯 |

## 🎨 우측 고정 위젯 구성 (전역 적용)

### ⭐ 위젯 전역 표시 규칙 (신규)
```javascript
// 모든 탭에서 우측 위젯 표시 (화상 스터디 제외)
const TABS_WITHOUT_SIDEBAR = ['video-call']

// 위젯 구성은 탭별로 동적 조정
const showSidebar = !pathname.includes('/video-call')
```

### 🎯 핵심 설계 원칙

1. **컨텍스트 유지**: 다른 탭으로 이동하지 않고도 중요 정보 확인
2. **빠른 액션**: 자주 사용하는 기능에 원클릭 접근
3. **시각적 피드백**: 진행 상황을 한눈에 파악
4. **인지 부하 최소화**: 필요한 정보만 간결하게 표시
5. **공간 효율성**: FHD 환경에서 좌우 여백 최소화

## 📱 반응형 설계

### Desktop (1280px+)
- **2컬럼 레이아웃**: 메인 70% + 사이드바 30%
- 사이드바 `position: sticky` (스크롤 시 고정)
- 컨테이너 `max-width: 1400px` (중앙 정렬)

### Tablet (768-1279px)
- **2컬럼 레이아웃**: 메인 65% + 사이드바 35%
- 사이드바 일부 위젯 숨김 (빠른 링크 등)
- 위젯 내부 텍스트 축약

### Mobile (<768px)
- **1컬럼 레이아웃**: 사이드바를 메인 콘텐츠 아래로 이동
- 순서: 주요 콘텐츠 → 빠른 액세스 → 통계 → 멤버
- 위젯을 아코디언으로 접을 수 있도록 처리

## 🔧 구현 참고사항

### 레이아웃 컴포넌트 구조
```jsx
/studies/[studyId]/layout.jsx
  └─ <StudyLayout>
      ├─ <StudyHeader /> (뒤로가기, 스터디 정보, 탭)
      └─ <div className={showSidebar ? 'withSidebar' : 'fullWidth'}>
          ├─ <main className="mainContent">
          │   {children}
          └─ {showSidebar && <StudySidebar />}

/components/study/sidebar/
  ├─ StudySidebar.jsx (메인 컨테이너)
  ├─ StatsWidget.jsx (통계)
  ├─ MembersWidget.jsx (멤버)
  ├─ QuickAccessWidget.jsx (빠른 액세스)
  ├─ InfoWidget.jsx (태그 & 정보)
  └─ QuickLinksWidget.jsx (빠른 링크)
```

### 우측 위젯 전역 표시 제어 로직 ⭐
```jsx
// layout.jsx
const pathname = usePathname()

// 화상 스터디만 위젯 숨김, 나머지 모든 탭에서 표시
const showSidebar = !pathname.includes('/video-call')

return (
  <div className={styles.studyContainer}>
    <StudyHeader />
    <div className={styles.withSidebar}>
      <main className={styles.mainContent}>
        {children}
      </main>
      {showSidebar && (
        <aside className={styles.sidebar}>
          <StudySidebar studyId={studyId} currentTab={currentTab} />
        </aside>
      )}
    </div>
  </div>
)
```

### 위젯 동적 조정 로직
```jsx
// StudySidebar.jsx
const StudySidebar = ({ studyId, currentTab }) => {
  return (

      <UpcomingEventsWidget studyId={studyId} highlight={currentTab === 'calendar'} />

    </div>
  )
}
```

## 💡 핵심 인사이트

1. **항상 보이는 정보 = 놓치지 않는 정보**
   - 고정 위젯으로 중요 정보를 지속 노출

2. **원클릭 액션 = 마찰 없는 UX**
   - 빠른 액션 버튼으로 핵심 기능 즉시 접근

3. **실시간 피드백 = 커뮤니티 활성화**
   - 온라인 멤버 표시로 소통 욕구 증가

4. **긴급도 시각화 = 우선순위 명확화**
   - D-day, 색상 코딩으로 중요도 전달

5. **컨텍스트 유지 = 효율성 극대화**
   - 탭 전환 없이 작업 완료 가능

6. **전역 표시 = 일관된 경험**
   - 모든 탭에서 동일한 위젯으로 학습 곡선 최소화
