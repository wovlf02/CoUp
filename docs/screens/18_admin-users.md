## 🎨 주요 기능

### 1. 필터 및 검색
- 상태 필터: 전체, 활성, 정지, 탈퇴
- 검색: 이름, 이메일

### 2. 사용자 테이블
**컬럼**:
- ID
- 프로필 이미지
- 이름
- 이메일
- 가입일
- 마지막 활동
- 상태 (활성/정지/탈퇴)
- 액션 (상세보기, 정지, 삭제)

### 3. 액션 버튼
- **상세보기**: 사용자 정보 모달
- **계정 정지**: 확인 후 정지
- **계정 삭제**: 확인 후 영구 삭제
