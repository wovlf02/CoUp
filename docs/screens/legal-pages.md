## 디자인 사양

### 레이아웃
- **최대 너비**: 800px (가독성 최적화)
- **중앙 정렬**
- **패딩**: 좌우 24px (모바일), 40px (데스크톱)
- **배경**: white

### 타이포그래피
```css
/* 페이지 제목 */
.pageTitle {
  font-size: 36px;
  font-weight: 700;
  color: var(--gray-900);
  margin-bottom: 8px;
}

/* 영문 부제 */
.pageSubtitle {
  font-size: 18px;
  font-weight: 400;
  color: var(--gray-500);
  margin-bottom: 24px;
}

/* 최종 수정일 */
.lastUpdated {
  font-size: 14px;
  color: var(--gray-600);
  margin-bottom: 40px;
}

/* 조항 제목 */
.articleTitle {
  font-size: 24px;
  font-weight: 700;
  color: var(--gray-900);
  margin-top: 48px;
  margin-bottom: 16px;
}

/* 본문 */
.content {
  font-size: 16px;
  line-height: 1.8;
  color: var(--gray-700);
}
```

### 목차 (Table of Contents)
- 스티키 사이드바 (데스크톱)
- 접을 수 있는 드롭다운 (모바일)
- 클릭 시 해당 조항으로 스크롤
- 현재 위치 하이라이트

## 공통 컴포넌트

### LegalHeader
- 뒤로가기 버튼 (좌측)
- 홈으로 버튼 (우측)
- 페이지 제목
- 최종 수정일

### TableOfContents
- 자동 생성된 목차
- 스크롤 위치 추적
- 현재 섹션 하이라이트

### LegalContent
- Markdown 형식의 법적 문서
- 조항 자동 넘버링
- 리스트 스타일링

## 접근성 (Accessibility)

1. **시맨틱 HTML**
   - `<article>` 태그 사용
   - 제목 계층 구조 (`h1` → `h2` → `h3`)

2. **키보드 네비게이션**
   - Tab 키로 목차 탐색
   - Enter로 섹션 이동

3. **스크린 리더**
   - `aria-label` 추가
   - 조항 번호 읽기 지원

4. **고대비 모드**
   - 충분한 색상 대비 (WCAG AA 기준)
