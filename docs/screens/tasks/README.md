# 할 일(Tasks) 기능 설계

> **네비게이션 탭**: 📋 할 일  
> **기본 라우트**: `/tasks`  
> **목적**: 모든 스터디의 할 일을 통합 관리

---

## 📌 주요 기능

### 1. 통합 할 일 관리
- 참여 중인 모든 스터디의 할 일을 한 곳에서 확인
- 마감일 기준 자동 정렬 및 그룹화
- 긴급(24시간 이내), 이번 주(7일 이내), 나중에(7일 이후) 구분

### 2. 필터링 & 정렬
- **스터디 필터**: 전체 / 특정 스터디
- **상태 필터**: 전체 / 미완료 / 완료
- **정렬**: 마감일순 / 최신순 / 스터디별 / 완료율순

### 3. 빠른 완료 처리
- 체크박스 클릭으로 즉시 완료 처리
- 낙관적 UI 업데이트
- Toast 알림 표시

### 4. 할 일 상세 모달
- 상세 설명 및 첨부 파일
- 멤버별 완료 현황
- 댓글 기능 (멤버 소통)
- 완료 처리 / 수정 / 삭제

### 5. 우측 위젯
- 오늘의 할 일
- 이번 주 진행률
- 스터디별 할 일 현황
- 이번 주 달성률
- 할 일 관리 팁

## 🎨 레이아웃 특징

### 3컬럼 구조
- 좌측 네비게이션: 12% (240px)
- 메인 콘텐츠: 58% (1100px)
- 우측 위젯: 30% (280px)

### 할 일 카드
- 긴급: 빨간색 배경 + 테두리
- 일반: 흰색 배경
- 완료: 회색 배경 + 투명도
