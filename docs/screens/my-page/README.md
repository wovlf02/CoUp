# 마이페이지(My Page) 기능 설계

> **네비게이션 탭**: 👤 마이페이지  
> **기본 라우트**: `/me`  
> **목적**: 사용자 프로필 및 계정 관리

---

## 📌 주요 기능

### 1. 프로필 관리
- 프로필 이미지 변경 (업로드)
- 이름 수정 (2-50자)
- 자기소개 수정 (최대 200자)
- 가입 정보 표시 (이메일, 가입일, 가입 방법)

### 2. 참여 스터디 현황
- 참여 중인 모든 스터디 목록
- 역할 배지 표시 (OWNER/ADMIN/MEMBER/PENDING)
- 멤버 수, 마지막 활동 시간
- [이동하기] 버튼으로 빠른 이동
- [전체 보기] → `/my-studies`

### 3. 활동 통계
- 총 참여 스터디 수
- 이번 주 활동 요약
- 완료한 할 일 수
- 출석률

### 4. 계정 관리
- **로그아웃**: 현재 기기에서 로그아웃
- **계정 삭제**: 
  - 경고 다이얼로그
  - "삭제" 텍스트 입력 확인
  - 모든 데이터 영구 삭제
  - OWNER인 스터디도 삭제

### 5. 알림 설정 (Phase 2)
- 푸시 알림 ON/OFF
- 이메일 알림 ON/OFF
- 알림 유형별 설정

### 6. 테마 설정 (Phase 2)
- 라이트 모드 / 다크 모드

## 🎨 레이아웃 특징

### 2컬럼 구조
- 좌측 네비게이션: 12% (240px)
- 메인 콘텐츠: 88%

### 섹션 구성
1. **프로필 섹션**: 이미지 + 기본 정보
2. **내 정보 수정**: 이름, 자기소개 폼
3. **참여 중인 스터디**: 스터디 목록 (최대 4개 표시)
4. **계정 관리**: 로그아웃, 계정 삭제

## ⚠️ 계정 삭제 확인 프로세스

1. [계정 삭제] 버튼 클릭
2. 확인 다이얼로그 표시:
   ```
   정말로 계정을 삭제하시겠습니까?
   
   삭제되는 데이터:
   - 프로필 정보
   - 참여 중인 모든 스터디 (OWNER인 스터디 포함)
   - 작성한 공지사항, 파일, 댓글 등
   
   이 작업은 되돌릴 수 없습니다.
   삭제하려면 "삭제"를 입력하세요:
   [입력창]
   
   [취소]  [계정 삭제]
   ```
3. "삭제" 텍스트 정확히 입력 확인
4. 계정 삭제 API 호출
5. 로그아웃 + 랜딩 페이지로 이동
