## 🎨 상세 설계

### 전체 레이아웃 (로그인과 동일)
```
배경: gray-50
중앙 카드: white, 그림자
카드 크기: 480px × auto (Desktop)
패딩: 48px (Desktop), 24px (Mobile)
둥근 모서리: 16px
```

### 헤더
```
[Logo] CoUp
새로운 여정을 시작하세요
```

**차이점**:
- 제목: "새로운 여정을 시작하세요" (로그인과 다른 메시지)

### 소셜 가입 버튼
```
┌────────────────────────────────────┐
│  [G]  Google로 시작하기            │
└────────────────────────────────────┘

┌────────────────────────────────────┐
│  [GitHub]  GitHub로 시작하기       │
└────────────────────────────────────┘
```

**동일한 스타일** (로그인 페이지와 동일)

### 약관 동의 안내
```
회원가입 시 [이용약관] 및 [개인정보처리방침]에
동의하는 것으로 간주됩니다.
```

**스타일**:
- 텍스트: gray-500 (text-xs, 12px)
- 링크: primary-500, Underline
- 중앙 정렬
- 버튼 아래 24px 간격

### 로그인 링크
```
이미 계정이 있으신가요? [로그인]
```

**클릭** → `/sign-in`

---

## 🔄 회원가입 흐름

### 첫 가입 시
1. 소셜 로그인 성공
2. **프로필 완성 페이지로 이동** (`/onboarding`)
   - 닉네임 입력 (필수)
   - 관심 카테고리 선택 (선택)
   - 자기소개 입력 (선택)
3. 완료 후 대시보드로 이동

### 이미 가입된 경우
1. 소셜 로그인 성공
2. 바로 대시보드로 이동
