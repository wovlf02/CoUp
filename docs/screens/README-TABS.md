# 화면 설계 문서 - 네비게이션 탭 기능

> CoUp 프로젝트의 네비게이션 탭 (할 일, 알림, 마이페이지) 화면 설계 문서

---

## 📁 문서 구조

```
docs/screens/
├── tasks/              # 할 일 탭
│   ├── README.md       # 기능 개요
│   └── main.md         # 메인 화면 상세 설계
├── notifications/      # 알림 탭
│   ├── README.md       # 기능 개요
│   └── main.md         # 메인 화면 상세 설계
└── my-page/           # 마이페이지 탭
    ├── README.md       # 기능 개요
    └── main.md         # 메인 화면 상세 설계
```

---

## 📋 할 일 (Tasks)

### 주요 기능
- 모든 스터디의 할 일을 통합 관리
- 마감일 기준 자동 그룹화 (긴급/이번주/나중에)
- 빠른 완료 체크 및 진행 상황 추적
- 스터디별 필터링 및 정렬
- 할 일 상세 모달 (댓글, 첨부파일)

### 화면 목록
- **메인 화면** (`/tasks`) - [상세 보기](tasks/main.md)

### 레이아웃
- 3컬럼 구조: Nav (12%) + Content (58%) + Widget (30%)
- 할 일 카드: 긴급도별 색상 구분
- 우측 위젯: 오늘의 할일, 진행률, 스터디별 현황, 달성률

---

## 🔔 알림 (Notifications)

### 주요 기능
- 스터디 활동 알림 실시간 수신
- 9가지 알림 유형 (가입승인, 공지, 파일, 일정, 할일, 멤버, 강퇴, 댓글, 멘션)
- 읽음/읽지않음 필터
- 알림 클릭 시 자동 읽음 처리 및 해당 페이지 이동
- WebSocket 기반 실시간 푸시

### 화면 목록
- **알림 목록** (`/notifications`) - [상세 보기](notifications/main.md)

### 레이아웃
- 2컬럼 구조: Nav (12%) + Content (88%)
- 알림 카드: 유형별 색상 배지
- 페이지네이션: 20개씩

---

## 👤 마이페이지 (My Page)

### 주요 기능
- 프로필 이미지 및 정보 관리
- 이름, 자기소개 수정
- 참여 중인 스터디 현황
- 계정 관리 (로그아웃, 계정 삭제)

### 화면 목록
- **마이페이지 메인** (`/me`) - [상세 보기](my-page/main.md)

### 레이아웃
- 2컬럼 구조: Nav (12%) + Content (88%)
- 섹션별 카드: 프로필, 내 정보, 참여 스터디, 계정 관리
- 최대 너비: 800px (중앙 정렬)

---

## 🎨 공통 디자인 원칙

### CSS 스타일
- **인라인 스타일 금지**: 모든 스타일은 CSS Module로 분리
- **BEM 네이밍**: `.section__element--modifier` 형식 권장
- **색상 시스템**: Tailwind CSS 색상 팔레트 사용
- **반응형**: Desktop → Tablet → Mobile 순차 대응

### 컴포넌트 구조
```
ComponentName.jsx      # 로직
ComponentName.module.css  # 스타일
```

### 상태 관리
- React Query: 서버 상태
- Zustand: 클라이언트 상태
- Context API: 전역 설정
