**고정 공지** (상단 고정):
- 📌 아이콘 + [고정] 배지
- 배경: primary-50 (연한 파란색)
- 테두리: primary-200

**일반 공지**:
- 📝 아이콘
- 배경: white
- 테두리: gray-200

**카드 내용**:
1. 제목 (text-lg, Bold, gray-900)
   - 클릭 → 공지 상세 페이지
2. 작성자 + 작성 시간 (text-sm, gray-600)
3. 액션 버튼 (우측, 관리자만)
   - [수정] [삭제]

## 🎬 인터랙션

### 공지 작성
```javascript
const handleCreate = async (data) => {
  await api.post(`/api/v1/studies/${studyId}/notices`, {
    title: data.title,
    content: data.content,
    isPinned: data.isPinned
  })
  
  toast.success('공지사항이 작성되었습니다!')
  closeModal()
  refetch()
}
```

### 공지 삭제
```javascript
const handleDelete = async (noticeId) => {
  if (!confirm('정말 삭제하시겠습니까?')) return
  
  await api.delete(`/api/v1/studies/${studyId}/notices/${noticeId}`)
  
  toast.success('공지사항이 삭제되었습니다!')
  refetch()
}
```
