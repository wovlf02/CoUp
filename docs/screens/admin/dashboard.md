## 🔐 권한 확인

### 접근 제어
```javascript
// middleware.js
if (path.startsWith('/admin')) {
  const user = await getUser(session)
  
  if (!user || user.role !== 'SYSTEM_ADMIN') {
    return redirect('/dashboard')
  }
}
```

### 역할 구조
```
- USER: 일반 사용자
- STUDY_ADMIN: 스터디 관리자 (그룹장)
- SYSTEM_ADMIN: 시스템 관리자 (전체 관리)
```
