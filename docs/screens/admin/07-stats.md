# 통계 분석 화면

> **작성일**: 2025-11-26

---

## 📋 레이아웃

```
┌─────────────────────────────────────────────────────────┐
│ 📊 통계 분석                                             │
├─────────┬─────────┬─────────┬─────────┐                │
│ 전체    │ 활성    │ 신규    │ 미처리  │                │
│ 사용자  │ 스터디  │ 가입    │ 신고    │                │
├─────────────────────────────────────────┤                │
│ 📈 사용자 분석                          │                │
│ - 사용자 증가 추이 (라인 차트)          │                │
│ - 가입 방법별 분포 (파이 차트)          │                │
├─────────────────────────────────────────┤                │
│ 📚 스터디 분석                          │                │
│ - 스터디 생성 추이 (라인 차트)          │                │
│ - 카테고리별 분포 (바 차트)             │                │
├─────────────────────────────────────────┤                │
│ 📝 활동 분석                            │                │
│ - 일별 활동 추이 (라인 차트)            │                │
│ - 활동 유형별 분포 (파이 차트)          │                │
└─────────────────────────────────────────┘
```

---

## 📈 차트 종류

### 1. 라인 차트 (Recharts)
- 사용자 증가 추이
- 스터디 생성 추이
- 활동 추이

### 2. 바 차트
- 카테고리별 스터디 수

### 3. 파이 차트
- 가입 방법별 분포
- 활동 유형별 분포
