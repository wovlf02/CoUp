## 🎨 주요 차트

### 1. 사용자 증가 추이 (라인 차트)
- 신규 가입자
- 활성 사용자
- 탈퇴 사용자

### 2. 스터디 카테고리 분포 (파이 차트)
- 프로그래밍
- 취업준비
- 자격증
- 어학
- 기타

### 3. 활동 통계 (바 차트)
- 일별 채팅 메시지 수
- 파일 업로드 수
- 공지사항 작성 수

### 4. 주요 지표
- DAU (Daily Active Users)
- MAU (Monthly Active Users)
- 평균 체류 시간
- 이탈률

---

## 📊 데이터 수집

### API
```
GET /api/v1/admin/analytics?period=30d

Response:
{
  users: { total, new, active, churn },
  studies: { total, byCategory },
  activities: { messages, files, notices },
  metrics: { dau, mau, avgDuration, bounceRate }
}
```
