## 🎨 주요 기능

### 1. 필터 및 검색
- 상태: 전체, 공개, 비공개
- 카테고리 필터
- 검색: 스터디명

### 2. 스터디 테이블
**컬럼**:
- ID
- 카테고리 이모지
- 스터디명
- 그룹장
- 멤버 수 (현재/최대)
- 공개 여부
- 생성일
- 액션 (상세, 숨김, 삭제)

### 3. 액션
- **상세보기**: 스터디 상세 페이지
- **숨김 처리**: 검색에서 제외 (규칙 위반 시)
- **강제 삭제**: 스팸 또는 악의적 스터디 삭제
