# 화면 설계 - 공통 컴포넌트

> **파일**: components.md
> **목적**: 관리자 시스템 전반에서 재사용되는 핵심 UI 컴포넌트의 명세와 동작을 정의하여, 일관된 사용자 경험을 보장하고 개발 생산성을 높입니다.

---

## 1. AdminTable

-   **설명**: 데이터 목록을 표시하는 표준 테이블 컴포넌트. `react-table` 또는 `TanStack Table` 라이브러리를 기반으로 구현하는 것을 권장합니다.
-   **주요 기능 (Props로 제어)**:
    -   `columns`: 테이블의 컬럼 정의. (헤더 텍스트, 데이터 키, 셀 렌더링 방식 등)
    -   `data`: 테이블에 표시할 데이터 배열.
    -   `showPagination`: 페이지네이션 표시 여부. (boolean)
    -   `showSelection`: 행 선택 체크박스 표시 여부. (boolean)
    -   `onRowClick`: 행 클릭 시 실행될 콜백 함수.
-   **UX**:
    -   컬럼 헤더를 클릭하여 해당 컬럼을 기준으로 오름차순/내림차순 정렬이 가능해야 합니다.
    -   많은 양의 텍스트가 있는 셀의 경우, 텍스트를 일부만 보여주고 `...`으로 말줄임 처리한 뒤, 마우스를 올리면 전체 내용을 툴팁으로 보여줍니다.

---

## 2. AdminModal

-   **설명**: 특정 액션을 수행하거나 추가 정보를 표시하기 위해 화면 위에 오버레이되는 모달(Modal) 컴포넌트.
-   **주요 기능 (Props로 제어)**:
    -   `isOpen`: 모달의 열림/닫힘 상태.
    -   `onClose`: 모달을 닫는 함수. (배경 클릭 또는 ESC 키로도 닫혀야 함)
    -   `title`: 모달의 제목.
    -   `children`: 모달의 본문 내용.
    -   `footer`: '확인', '취소' 등 액션 버튼들이 위치하는 영역.
-   **UX**:
    -   모달이 열리면 뒷 배경은 흐리게(dimmed) 처리하고, 뒷 배경의 스크롤은 방지하여 사용자가 모달에 집중하도록 합니다.
    -   위험한 액션(예: 삭제)을 수행하는 모달의 '확인' 버튼은 빨간색으로 표시하여 경고의 의미를 전달합니다.

---

## 3. StatCard (통계 카드)

-   **설명**: 대시보드 등에서 핵심 지표(KPI)를 시각적으로 강조하여 보여주는 카드 컴포넌트.
-   **주요 기능 (Props로 제어)**:
    -   `title`: 지표 이름.
    -   `value`: 지표 값 (숫자 또는 문자열).
    -   `icon`: 지표를 상징하는 아이콘.
    -   `trend`: `up`, `down`, `neutral` 등 추세. 값에 따라 화살표 아이콘과 색상(up: green, down: red)이 결정됩니다.
    -   `trendValue`: 증감률 (예: "+1.5%")
    -   `onClick`: 카드 클릭 시 실행될 함수.
-   **UX**:
    -   값을 로딩하는 동안에는 숫자 대신 스켈레톤(skeleton) UI를 보여주어 화면의 깜빡임을 방지하고 사용성을 개선합니다.

---

## 4. AdminBadge (배지)

-   **설명**: 사용자의 상태, 신고 우선순위, 역할 등 특정 카테고리 값을 나타내는 작은 텍스트 라벨.
-   **주요 기능 (Props로 제어)**:
    -   `variant`: `success`, `danger`, `warning`, `info`, `neutral` 등 배지의 색상을 결정하는 값.
    -   `children`: 배지 안에 표시될 텍스트.
-   **UX**:
    -   `status: 'ACTIVE'`는 초록색(`success`), `status: 'SUSPENDED'`는 회색(`neutral`), `priority: 'URGENT'`는 빨간색(`danger`)과 같이, 값의 의미와 색상을 일관되게 매칭하여 사용자가 텍스트를 읽지 않고도 의미를 빠르게 파악할 수 있도록 합니다.

---

## 5. FilterPanel

-   **설명**: 테이블이나 리스트의 데이터를 필터링하기 위한 여러 컨트롤(드롭다운, 날짜 선택 등)을 모아놓은 패널.
-   **주요 기능 (Props로 제어)**:
    -   `filters`: 표시할 필터의 종류와 옵션을 정의하는 객체 배열.
    -   `onFilterChange`: 필터 값이 변경되었을 때, 변경된 필터 값을 인자로 받아 실행되는 콜백 함수.
-   **UX**:
    -   필터 값이 변경되면 즉시 결과가 업데이트되어야 합니다.
    -   '초기화' 버튼을 제공하여 모든 필터 조건을 한 번에 제거할 수 있도록 합니다.
