# 스터디 목록 페이지

> 스터디 조회, 검색 및 모더레이션

## 📁 파일 구조

```
src/app/admin/studies/
├── page.jsx
└── _components/
    ├── StudyTable.jsx         # 스터디 테이블 (~200줄)
    ├── StudyFilters.jsx       # 필터 (~150줄)
    └── StudyActions.jsx       # 액션 (~100줄)
```

## 주요 기능

### 1. 스터디 테이블
- 스터디 이름, 카테고리
- 스터디장 정보
- 멤버 수, 활동 점수
- 상태 (정상/숨김/종료)
- 신고 수, 경고 수

### 2. 필터링
- 카테고리별
- 상태별 (정상/숨김/종료/삭제)
- 공개/비공개
- 신고 여부
- 날짜 범위

### 3. 정렬
- 최근 생성순
- 멤버 수 순
- 활동 점수 순
- 신고 수 순

### 4. 빠른 액션
- 스터디 숨김/해제
- 스터디 종료
- 추천 스터디 등록
- 스터디 삭제

## 구현 패턴

사용자 목록과 유사한 구조:
- Server Component 페이지
- Client Component 테이블
- 검색/필터는 URL 쿼리 파라미터
- 페이지네이션 및 정렬
- 일괄 작업 지원
