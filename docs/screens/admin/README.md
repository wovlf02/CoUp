## 🎯 화면별 개요

### 1. 대시보드 (`/admin`)
- 핵심 지표 카드 (4개)
- 차트 (가입자 추이, 신고 분포)
- 최근 활동 타임라인
- 긴급 알림

### 2. 사용자 관리 (`/admin/users`)
- 사용자 목록 (테이블)
- 검색/필터
- 사용자 상세 모달
- 제재 모달 (경고/정지)

### 3. 스터디 관리 (`/admin/studies`)
- 스터디 목록
- 스터디 상세 (4개 탭)
- 제재 모달
- 추천 스터디 관리

### 4. 신고 관리 (`/admin/reports`)
- 신고 목록 (우선순위 표시)
- 신고 상세 사이드 패널
- 처리 모달
- 댓글 시스템

### 5. 콘텐츠 검열 (`/admin/moderation`)
- 차단 로그
- 검열 대기 목록
- 콘텐츠 검토 모달
- 금지어 관리 (SYSTEM_ADMIN)

### 6. 시스템 설정 (`/admin/settings`)
- 플랫폼 설정 폼
- 이메일 템플릿 에디터
- 이용약관 편집기
- 관리자 권한 관리

### 7. 분석 (`/admin/analytics`)
- 통계 대시보드
- 인터랙티브 차트
- 리포트 생성 폼
- 리포트 히스토리

## 🧩 핵심 컴포넌트

### 1. AdminTable
재사용 가능한 데이터 테이블
- 정렬
- 필터링
- 페이지네이션
- 선택
- 액션 버튼

### 2. AdminModal
관리자 액션 모달
- 확인 모달
- 폼 모달
- 상세 보기 모달

### 3. AdminCard
통계 카드
- 숫자 표시
- 증감률
- 아이콘
- 링크

### 4. AdminChart
차트 컴포넌트 (Recharts)
- 선 그래프
- 막대 그래프
- 도넛 차트

### 5. AdminBadge
상태 배지
- 사용자 상태
- 신고 우선순위
- 역할
