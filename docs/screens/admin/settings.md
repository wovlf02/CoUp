## 🎨 주요 설정

### 1. 서비스 설정
- **서비스 상태**: 정상 운영 / 점검 모드
- **회원가입 허용**: On/Off
- **공개 스터디 생성**: On/Off

### 2. 제한 설정
- **스터디 생성 제한**: 사용자당 최대 개수
- **파일 업로드 크기**: MB 단위
- **멤버 수 제한**: 스터디당 최대 인원
- **메시지 길이 제한**: 최대 글자 수

### 3. 관리자 계정
- 관리자 목록
- 역할: 슈퍼 관리자, 모더레이터
- 관리자 추가/삭제

### 4. 백업 및 로그
- **데이터 백업**: 수동/자동 백업
- **로그 보기**: 시스템 로그 다운로드
- **에러 로그**: 최근 에러 확인

---

## 📋 설정 저장

### API
```
PATCH /api/v1/admin/settings

Body:
{
  service: { status, signupEnabled },
  limits: { maxStudies, maxFileSize },
  admins: [{ email, role }]
}
```
