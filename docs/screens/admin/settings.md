## 🎨 섹션별 상세 설계

### 1. 탭 네비게이션 (상단)

```
┌──────────────────────────────────────────────────────────────┐
│ [서비스 설정] [제한 설정] [관리자 계정] [백업 및 로그]        │
└──────────────────────────────────────────────────────────────┘
```

**탭 구성**:
1. 서비스 설정 - 서비스 상태, 기능 on/off
2. 제한 설정 - 각종 제한 값
3. 관리자 계정 - 관리자 추가/삭제
4. 백업 및 로그 - 데이터 백업, 시스템 로그
