## 🎨 주요 기능

### 1. 신고 유형
- 스팸
- 욕설/비방
- 부적절한 콘텐츠
- 저작권 침해
- 기타

### 2. 신고 테이블
**컬럼**:
- ID
- 유형 (배지)
- 신고 대상 (스터디/사용자)
- 신고자
- 신고 일시
- 처리 상태
- 액션 (상세보기, 처리하기)

### 3. 신고 처리
- **상세보기**: 신고 내용 및 증거 확인
- **경고**: 대상에게 경고 발송
- **계정 정지**: 사용자 계정 정지
- **콘텐츠 삭제**: 스터디/게시물 삭제
- **신고 기각**: 부당 신고 기각
