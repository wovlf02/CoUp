## ? 온보딩 기간 설정

### 기본 기간: 7일
```javascript
const isOnboardingPeriod = () => {
  const joinDate = new Date(membership.createdAt)
  const now = new Date()
  const diff = now - joinDate
  const days = diff / (1000 * 60 * 60 * 24)
  
  return days <= 7
}
```

### 조기 종료 옵션
- "다시 보지 않기" 버튼
- 모든 단계 완료 시 자동 종료

### 재표시 옵션
- 설정에서 "온보딩 다시 보기" 제공
