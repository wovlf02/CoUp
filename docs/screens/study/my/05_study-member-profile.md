## ? 인터랙션

### DM 보내기
```javascript
const handleSendDM = () => {
  router.push(`/messages/${userId}`)
  // 또는 모달로 처리
}
```

### 멤버 평가하기
```javascript
const handleRate = async (ratings) => {
  await api.post(`/api/v1/studies/${studyId}/members/${userId}/rate`, {
    leadership: ratings.leadership,
    sincerity: ratings.sincerity,
    collaboration: ratings.collaboration,
    knowledge: ratings.knowledge,
    comment: ratings.comment
  })
  
  toast.success('평가가 완료되었습니다!')
  refetch()
}
```

## ? 뱃지 시스템

### 출석 관련
- ? **3일 연속**: 3일 연속 출석
- ? **7일 연속**: 7일 연속 출석
- ? **30일 연속**: 30일 연속 출석
- ? **100% 출석**: 한 달 개근

### 기여도 관련
- ? **MVP**: 주간 활동 1위
- ? **파일 공유왕**: 파일 10개 이상 공유
- ? **채팅 활발**: 채팅 100개 이상
- ? **할일 마스터**: 할일 완료율 95% 이상

### 특별 기여
- ? **멘토**: 신입 멤버 온보딩 3회 이상
- ? **화상 리더**: 화상 스터디 진행 5회 이상
- ? **공지 작성**: 공지 10개 이상 작성

## ? 반응형 설계

### Desktop (1280px+)
- 2컬럼 레이아웃 (70% + 30%)
- 탭 수평 배치

### Tablet (768-1279px)
- 2컬럼 레이아웃 축소
- 우측 위젯 일부 숨김

### Mobile (<768px)
- 1컬럼 레이아웃
- 우측 위젯 하단 이동
- 탭 스크롤 가능

## ? UX 개선 포인트

1. **투명성**
   - 모든 활동이 기록되고 표시됨
   - 멤버 간 기여도 비교 가능

2. **동기부여**
   - 뱃지 시스템으로 성취감 제공
   - 주간 MVP 선정
   - 활동 지수로 경쟁 요소

3. **피드백**
   - 멤버 간 익명 평가
   - 긍정적인 피드백 문화

4. **소통 촉진**
   - DM 바로가기
   - 최근 활동으로 대화 주제 제공

5. **공정성**
   - 객관적인 지표 사용
   - 자동 계산으로 편향 제거
