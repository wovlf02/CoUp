# 파일 관리 (Study Files)

> **화면 ID**: `STUDY-FILES-01`  
> **라우트**: `/my-studies/[studyId]/files`  
> **목적**: 스터디 파일 업로드, 관리, 공유  
> **접근 권한**: MEMBER+ (읽기/다운로드), MEMBER+ (업로드), ADMIN+ (삭제)  
> **렌더링**: CSR (실시간 업데이트)

## 📥 파일 다운로드

### 1. 단일 파일 다운로드

```javascript
const handleDownload = async (fileId, fileName) => {
  try {
    setDownloading(true)
    
    const response = await api.get(
      `/api/v1/my-studies/${studyId}/files/${fileId}/download`,
      { responseType: 'blob' }
    )
    
    // 파일 다운로드
    const url = window.URL.createObjectURL(new Blob([response.data]))
    const link = document.createElement('a')
    link.href = url
    link.setAttribute('download', fileName)
    document.body.appendChild(link)
    link.click()
    link.remove()
    
    // 다운로드 수 증가
    await api.post(`/api/v1/my-studies/${studyId}/files/${fileId}/increment-download`)
    
    toast.success('다운로드가 시작되었습니다')
  } catch (error) {
    toast.error('다운로드 중 오류가 발생했습니다')
  } finally {
    setDownloading(false)
  }
}
```

### 2. 일괄 다운로드 (ZIP)

```javascript
const handleBulkDownload = async (fileIds) => {
  if (fileIds.length === 0) {
    toast.error('다운로드할 파일을 선택해주세요')
    return
  }
  
  try {
    setDownloading(true)
    
    toast.info('파일을 압축하고 있습니다...')
    
    // 서버에서 ZIP 생성
    const response = await api.post(
      `/api/v1/my-studies/${studyId}/files/download-bulk`,
      { fileIds },
      { responseType: 'blob' }
    )
    
    // ZIP 다운로드
    const url = window.URL.createObjectURL(new Blob([response.data]))
    const link = document.createElement('a')
    link.href = url
    link.setAttribute('download', `files_${Date.now()}.zip`)
    document.body.appendChild(link)
    link.click()
    link.remove()
    
    toast.success(`${fileIds.length}개 파일이 다운로드되었습니다`)
  } catch (error) {
    toast.error('일괄 다운로드 중 오류가 발생했습니다')
  } finally {
    setDownloading(false)
  }
}
```

### 1. 파일 업로드

```javascript
const handleUpload = async (files) => {
  // 검증
  if (files.length > 10) {
    toast.error('한 번에 최대 10개까지 업로드 가능합니다')
    return
  }
  
  const oversizedFiles = files.filter(f => f.size > 50 * 1024 * 1024)
  if (oversizedFiles.length > 0) {
    toast.error('50MB 이하 파일만 업로드 가능합니다')
    return
  }
  
  // 저장 공간 확인
  const totalSize = files.reduce((sum, f) => sum + f.size, 0)
  if (storageUsed + totalSize > storageLimit) {
    toast.error('저장 공간이 부족합니다')
    return
  }
  
  try {
    setUploading(true)
    
    const formData = new FormData()
    files.forEach(file => formData.append('files', file))
    formData.append('folderId', currentFolderId || '')
    
    // 업로드 진행률 추적
    const response = await api.post(
      `/api/v1/my-studies/${studyId}/files/upload`,
      formData,
      {
        headers: { 'Content-Type': 'multipart/form-data' },
        onUploadProgress: (progressEvent) => {
          const percentCompleted = Math.round(
            (progressEvent.loaded * 100) / progressEvent.total
          )
          setUploadProgress(percentCompleted)
        }
      }
    )
    
    toast.success(`${files.length}개 파일이 업로드되었습니다`)
    
    // 목록 새로고침
    queryClient.invalidateQueries(['files', studyId])
  } catch (error) {
    toast.error('업로드 중 오류가 발생했습니다')
  } finally {
    setUploading(false)
    setUploadProgress(0)
  }
}
```

### 2. 폴더 생성

```javascript
const handleCreateFolder = async (folderName, description) => {
  if (!folderName || folderName.length < 2) {
    toast.error('폴더 이름은 2자 이상 입력해주세요')
    return
  }
  
  // 특수문자 검증
  if (!/^[가-힣a-zA-Z0-9\s_-]+$/.test(folderName)) {
    toast.error('폴더 이름에 특수문자는 사용할 수 없습니다')
    return
  }
  
  try {
    await api.post(`/api/v1/my-studies/${studyId}/folders`, {
      name: folderName,
      description,
      parentId: currentFolderId || null
    })
    
    toast.success('폴더가 생성되었습니다')
    
    queryClient.invalidateQueries(['files', studyId])
  } catch (error) {
    toast.error('폴더 생성 중 오류가 발생했습니다')
  }
}
```

const handleDeleteFiles = async (fileIds) => {
  const confirmed = confirm(
    `${fileIds.length}개 파일을 삭제하시겠습니까?\n` +
    '삭제된 파일은 휴지통으로 이동됩니다.'

  }
}
