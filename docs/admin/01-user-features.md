## 1. 인증 및 계정

### 1.1 회원가입
**화면**: `/sign-up`

**기능**:
- 이메일 + 비밀번호 가입
- 소셜 로그인 (Google, GitHub)
- 이용약관 동의 필수
- 이메일 인증 (선택)

**관리 필요 사항**:
- 부적절한 닉네임 필터링
- 스팸 계정 탐지
- 가입 통계 추적

### 1.2 로그인
**화면**: `/sign-in`

**기능**:
- 이메일/비밀번호 로그인
- 소셜 로그인
- 로그인 상태 유지
- 비밀번호 찾기

**관리 필요 사항**:
- 비정상 로그인 시도 모니터링
- 계정 정지 시 로그인 차단
- 로그인 로그 기록

## 2. 대시보드

**화면**: `/dashboard`

### 표시 내용
1. **통계 카드 (4개)**
   - 참여 스터디 수
   - 새 공지 수
   - 미완료 할일 수
   - 다가올 일정 수 (7일 이내)

2. **내 스터디 섹션**
   - 최대 3개 스터디 카드 표시
   - 이모지, 이름, 멤버수, 역할
   - 마지막 활동 시간
   - 빠른 액션 버튼 (채팅/공지/파일)

3. **최근 활동 타임라인**
   - 공지, 할일, 파일, 채팅, 일정
   - 최대 5개 표시
   - 타입별 배지

4. **우측 위젯**
   - 오늘의 할일 (5개)
   - 다가오는 일정 (3개)
   - 스터디 현황
   - 빠른 액션 버튼

**관리 필요 사항**:
- 전체 활동 모니터링
- 비활성 사용자 감지

## 3. 스터디 탐색

**화면**: `/studies`

### 기능
1. **검색**
   - 스터디 이름 검색
   - 태그 검색
   - 카테고리 검색

2. **필터링**
   - 카테고리 (프로그래밍, 취업, 어학 등)
   - 최대 인원
   - 공개 여부
   - 모집 상태

3. **정렬**
   - 최신순
   - 인기순 (멤버수)
   - 활동순

4. **스터디 카드**
   - 이모지, 이름, 설명
   - 카테고리, 태그
   - 멤버수/최대인원
   - 공개 여부, 자동 승인 여부
   - [둘러보기] 버튼

**관리 필요 사항**:
- 부적절한 스터디 숨김
- 우수 스터디 추천 마크
- 스팸 스터디 감지

---

## 4. 스터디 생성

**화면**: `/studies/create`

### 입력 필드
1. **기본 정보**
   - 스터디 이름 (필수, 2-50자)
   - 이모지 (선택)
   - 설명 (필수, 10-500자)

2. **카테고리**
   - 주 카테고리 (필수)
   - 서브 카테고리 (선택)

3. **모집 설정**
   - 최대 인원 (2-100명)
   - 공개 여부 (공개/비공개)
   - 자동 승인 여부 (ON/OFF)

4. **태그**
   - 최대 5개
   - 콤마로 구분

**관리 필요 사항**:
- 스터디 생성 제한 (스팸 방지)
- 부적절한 이름/설명 필터링
- 생성 통계

## 5. 내 스터디

**화면**: `/my-studies`

### 필터
- 전체
- 내가 만든 스터디 (OWNER)
- 관리 중인 스터디 (ADMIN)
- 참여 중 (MEMBER)
- 가입 대기 중 (PENDING)

### 스터디 카드
- 이모지, 이름
- 역할 배지 (OWNER/ADMIN/MEMBER/PENDING)
- 멤버 수
- 마지막 활동 시간
- 빠른 액션 버튼

**관리 필요 사항**:
- 스터디 참여 이력
- 비활성 스터디 정리

## 6. 스터디 콘텐츠

**기본 라우트**: `/my-studies/[studyId]`

### 6.1 개요 (대시보드)
**라우트**: `/my-studies/[studyId]`

**2컬럼 레이아웃**:
- **메인 (70%)**:
  - 스터디 정보 카드
  - 멤버 목록 (상위 6명)
  - 최근 활동 타임라인

- **우측 위젯 (30%)**:
  - 스터디 현황
  - 온라인 멤버
  - 급한 할일
  - 다가오는 일정

### 6.2 채팅
**라우트**: `/my-studies/[studyId]/chat`

**기능**:
- 실시간 WebSocket 채팅
- 무한 스크롤 (50개씩)
- 파일 첨부 (최대 10MB)
- 이모지 반응
- 읽음 표시
- 입력 중 표시

**관리 필요 사항**:
- 욕설/혐오 발언 필터링
- 스팸 메시지 차단
- 불법 파일 업로드 차단

### 6.3 공지사항
**라우트**: `/my-studies/[studyId]/notices`

**권한**:
- 작성/수정/삭제: OWNER, ADMIN
- 읽기: 모든 멤버

**기능**:
- 공지 목록 (최신순)
- 중요 공지 고정
- 조회수 표시
- 댓글 (선택)

**관리 필요 사항**:
- 부적절한 공지 삭제

### 6.4 파일
**라우트**: `/my-studies/[studyId]/files`

**기능**:
- 파일 업로드 (최대 50MB)
- 폴더 구조
- 파일 검색
- 다운로드 수 표시

**관리 필요 사항**:
- 불법 파일 차단
- 저작권 침해 파일 삭제
- 저장 용량 모니터링

### 6.5 캘린더
**라우트**: `/my-studies/[studyId]/calendar`

**기능**:
- 일정 생성/수정/삭제
- 월간/주간 뷰
- 구글 캘린더 연동 (선택)
- 일정 알림

### 6.6 할일
**라우트**: `/my-studies/[studyId]/tasks`

**기능**:
- 할일 생성/완료
- 담당자 지정
- 마감일 관리
- 우선순위 (높음/보통/낮음)
- 진행률 표시

### 6.7 멤버 관리 (OWNER, ADMIN만)
**라우트**: `/my-studies/[studyId]/members`

**기능**:
- 멤버 목록
- 역할 변경 (OWNER, ADMIN, MEMBER)
- 멤버 추방
- 가입 신청 승인/거절

**관리 필요 사항**:
- 멤버 분쟁 조정
- 강제 퇴출 요청 처리

### 6.8 설정 (OWNER, ADMIN만)
**라우트**: `/my-studies/[studyId]/settings`

**기능**:
- 스터디 정보 수정
- 공개 설정
- 모집 상태
- 스터디 삭제 (OWNER만)

## 7. 마이페이지

**화면**: `/me`

### 7.1 프로필
**표시**:
- 프로필 이미지
- 이름, 이메일
- 자기소개
- 가입일
- 참여 스터디 수

**수정 가능**:
- 프로필 이미지 (최대 5MB)
- 이름 (2-50자)
- 자기소개 (최대 200자)
- 비밀번호

**관리 필요 사항**:
- 부적절한 프로필 사진 검열
- 욕설/혐오 발언 자기소개 차단

### 7.2 내 스터디
- 참여 중인 스터디 목록
- 역할별 배지
- 전체보기 링크

### 7.3 활동 통계
**이번 주**:
- 완료한 할일 수
- 작성한 공지 수
- 업로드한 파일 수
- 참여한 채팅 횟수

**전체**:
- 총 참여 스터디 수
- 총 완료 할일 수
- 평균 출석률
- 가입 기간

**배지** (조건부):
- 🥇 스터디 마스터 (3개 이상)
- 🔥 연속 출석 7일
- ⭐ 할일 완료왕 (30개 이상)

### 7.4 계정 관리
- 로그아웃
- 계정 삭제 (확인 필요)

**관리 필요 사항**:
- 탈퇴 계정 데이터 처리

## 8. 알림

**화면**: `/notifications`

### 알림 유형
1. **STUDY_INVITE**: 스터디 초대
2. **JOIN_APPROVED**: 가입 승인
3. **JOIN_REJECTED**: 가입 거절
4. **NOTICE**: 새 공지
5. **FILE**: 새 파일 업로드
6. **TASK_ASSIGNED**: 할일 할당
7. **TASK_DUE**: 할일 마감 임박 (D-1)
8. **EVENT**: 일정 생성/변경
9. **EVENT_SOON**: 일정 시작 임박 (1시간 전)
10. **CHAT**: 새 채팅 메시지 (멘션 시)
11. **MEMBER_JOIN**: 새 멤버 가입
12. **MEMBER_LEAVE**: 멤버 탈퇴
13. **KICK**: 강퇴 당함
14. **ROLE_CHANGED**: 역할 변경
15. **SYSTEM**: 시스템 공지

### 기능
- 읽음/안읽음 표시
- 알림 삭제
- 알림 클릭 시 해당 페이지 이동
- 알림 설정 (카테고리별 ON/OFF)

**관리 필요 사항**:
- 시스템 공지 발송
- 긴급 알림 발송
- 알림 통계

## 9. 모달 목록

### 9.1 스터디 가입 신청
**트리거**: 스터디 상세 > [가입 신청] 버튼

**필드**:
- 자기소개 (필수, 10-200자)
- 동기 (필수, 10-200자)
- 레벨 선택 (초급/중급/고급)

### 9.2 멤버 추방 확인
**트리거**: 멤버 목록 > [추방] 버튼 (OWNER, ADMIN만)

**필드**:
- 추방 사유 (필수, 10-200자)
- 확인 버튼

### 9.3 스터디 삭제 확인
**트리거**: 스터디 설정 > [스터디 삭제] 버튼 (OWNER만)

**필드**:
- "삭제" 텍스트 정확히 입력
- 확인 버튼

### 9.4 파일 업로드
**트리거**: 파일 탭 > [업로드] 버튼

**기능**:
- 드래그 앤 드롭
- 폴더 선택
- 진행률 표시
- 다중 파일 업로드

### 9.5 일정 생성/수정
**트리거**: 캘린더 > 날짜 클릭 or [일정 추가]

**필드**:
- 제목 (필수)
- 날짜, 시간
- 장소 (선택)
- 색상 선택

### 9.6 할일 생성/수정
**트리거**: 할일 탭 > [할일 추가]

**필드**:
- 제목 (필수)
- 설명 (선택)
- 담당자 (선택)
- 마감일 (선택)
- 우선순위

### 9.7 역할 변경 확인
**트리거**: 멤버 목록 > [역할 변경] (OWNER만)

**필드**:
- 새 역할 선택 (ADMIN/MEMBER)
- 확인 버튼

### 9.8 계정 삭제 확인
**트리거**: 마이페이지 > [계정 삭제]

**필드**:
- 삭제 경고 안내
- "삭제" 텍스트 정확히 입력
- 확인 버튼

## 📊 사용자 역할 및 권한

### 플랫폼 역할
1. **USER**: 일반 사용자
   - 모든 일반 기능 사용

2. **ADMIN**: 관리자
   - 사용자/스터디 관리
   - 신고 처리
   - 통계 조회

3. **SYSTEM_ADMIN**: 시스템 관리자
   - 모든 관리자 권한
   - 시스템 설정
   - 관리자 권한 부여

### 스터디 역할
1. **OWNER**: 스터디장
   - 모든 권한
   - 스터디 삭제
   - 소유권 이전

2. **ADMIN**: 관리자
   - 멤버 관리
   - 공지 작성
   - 설정 변경
   - 스터디 삭제 불가

3. **MEMBER**: 일반 멤버
   - 채팅 참여
   - 파일 업로드/다운로드
   - 일정/할일 생성
   - 읽기 전용 (공지, 멤버 목록)

4. **PENDING**: 가입 대기
   - 읽기만 가능
   - 채팅/업로드 불가
