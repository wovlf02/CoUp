# 분석 대시보드 완전 명세

> CoUp 플랫폼의 관리자 분석 대시보드 최종 명세

## 개요

관리자가 플랫폼의 상태를 실시간으로 모니터링하고, 데이터 기반 의사결정을 내릴 수 있도록 지원하는 종합 분석 대시보드입니다.

## 핵심 지표

### 1. 사용자 지표
- DAU/WAU/MAU (일/주/월간 활성 사용자)
- 신규 가입자 추이
- 사용자 리텐션 (1일, 7일, 30일)
- 평균 세션 시간
- 정지/삭제 사용자 비율

### 2. 스터디 지표
- 전체 스터디 수 및 증가율
- 활성 스터디 비율
- 카테고리별 분포
- 평균 멤버 수
- 스터디 생명 주기 분석

### 3. 활동 지표
- 일일 메시지 수
- 파일 업로드 수
- 공지사항 작성 수
- 할일 완료율
- 출석 참여율

### 4. 모더레이션 지표
- 신고 접수/처리 건수
- 평균 처리 시간
- 제재 발생 건수
- 경고 누적 사용자 수

## 구현

```typescript
// API 엔드포인트
GET /api/admin/analytics/overview
GET /api/admin/analytics/users
GET /api/admin/analytics/studies
GET /api/admin/analytics/moderation
```

## UI 구성

- 대시보드 홈: 주요 지표 카드 + 그래프
- 사용자 분석: 상세 사용자 통계
- 스터디 분석: 스터디 트렌드 및 성과
- 모더레이션 분석: 신고 및 제재 통계
