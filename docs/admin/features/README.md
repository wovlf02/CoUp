# 관리자 기능 명세 모음

> **작성일**: 2025-11-26  
> **목적**: 영역별 관리자 기능 상세 명세

---

## 📚 문서 목록

### 1. [대시보드](./01-dashboard.md)
**라우트**: `/admin/dashboard`

**핵심 기능**:
- 핵심 지표 카드 (신규 가입자, 활성 사용자, 미처리 신고, 정지 사용자)
- 최근 관리 활동 타임라인
- 통계 그래프 (가입자 추이, 신고 분포)
- 긴급 알림 (미처리 신고, 스팸 계정 탐지 등)
- 빠른 액션 버튼

**권한**: ADMIN, SYSTEM_ADMIN

---

### 2. [사용자 관리](./02-user-management.md)
**라우트**: `/admin/users`

**핵심 기능**:
- 사용자 목록 조회 (검색, 필터, 정렬)
- 사용자 상세 정보
- 제재 실행:
  - 경고 발송
  - 정지 (1일/3일/7일/30일/영구)
  - 기능 제한 (채팅/스터디 생성/파일 업로드)
  - 정지 해제
- 제재 이력 조회
- 데이터 익스포트

**권한**: 
- ADMIN: 조회, 경고, 정지, 해제
- SYSTEM_ADMIN: ADMIN 권한 + 완전 삭제

---

### 3. [스터디 관리](./03-study-management.md)
**라우트**: `/admin/studies`

**핵심 기능**:
- 스터디 목록 조회
- 스터디 상세 (4개 탭: 개요, 콘텐츠, 멤버, 활동 로그)
- 제재 실행:
  - 숨김 (검색 제외)
  - 모집 중단
  - 강제 종료
  - 숨김 해제
- 추천 스터디 지정/관리
- 콘텐츠 삭제 (메시지, 파일)

**권한**:
- ADMIN: 조회, 제재, 추천
- SYSTEM_ADMIN: ADMIN 권한 + 복구

---

### 4. [신고 관리](./04-report-management.md)
**라우트**: `/admin/reports`

**핵심 기능**:
- 신고 목록 (검색, 필터, 우선순위)
- 신고 상세 정보
- 담당자 배정 (자동/수동)
- 처리 결정:
  - 승인 (제재 실행)
  - 기각
  - 보류
  - 에스컬레이션 (SYSTEM_ADMIN)
- 댓글 시스템
- 관련 신고 조회
- 신고 통계

**권한**: ADMIN, SYSTEM_ADMIN

---

### 5. [콘텐츠 검열](./05-content-moderation.md)
**라우트**: `/admin/moderation`

**핵심 기능**:
- 자동 필터링:
  - 금지어 조회 (ADMIN)
  - 금지어 관리 (SYSTEM_ADMIN)
  - 필터링 강도 설정
  - 자동 제재 규칙
- 차단 로그
- 수동 검열:
  - 검열 대기 목록
  - 콘텐츠 검토 (승인/삭제)
  - 오탐 처리 (복구)
- 검열 대시보드
- 파일 확장자 제한

**권한**:
- ADMIN: 조회, 수동 검열
- SYSTEM_ADMIN: ADMIN 권한 + 규칙 설정

---

### 6. [시스템 설정](./06-system-settings.md)
**라우트**: `/admin/settings`

**핵심 기능**:
- 플랫폼 설정:
  - 회원가입 제한
  - 스터디 생성 제한
  - 파일 업로드 제한
- 이메일 템플릿 관리
- 이용약관 관리 (버전 관리)
- 관리자 권한 관리:
  - 관리자 임명/해임
  - 활동 모니터링

**권한**: SYSTEM_ADMIN 전용

---

### 7. [분석 및 리포트](./07-analytics.md)
**라우트**: `/admin/analytics`

**핵심 기능**:
- 사용자 분석:
  - 가입자 추이
  - 활성 사용자
  - 코호트 분석
  - 사용자 세그먼트
- 스터디 분석:
  - 생성 추이
  - 카테고리별 분포
  - 활성도 분석
  - 인기 TOP 10
- 활동 분석:
  - 일별/요일별 활동
  - 시간대별 분포
  - 참여도 분석
- 리포트 생성:
  - 일간/주간/월간 리포트
  - PDF/Excel/HTML 출력
  - 자동 예약 (SYSTEM_ADMIN)

**권한**:
- ADMIN: 조회, 리포트 생성 (제한)
- SYSTEM_ADMIN: 모든 기능 + 민감 데이터

## 📊 구현 우선순위

### Phase 1: 핵심 기능 (필수) - 2주
1. ✅ 대시보드 (01)

3. ✅ 스터디 관리 (03) - 목록, 상세, 숨김, 종료
4. ✅ 신고 관리 (04) - 목록, 처리 (승인/기각)

### Phase 3: 분석 및 개선 - 1주
6. ✅ 분석 및 리포트 (07) - 기본 통계, 리포트 생성

**목표**: 데이터 기반 의사결정

### Phase 4: 고급 기능 - 1주
7. ✅ 시스템 설정 (06) - 플랫폼 설정, 템플릿, 약관
