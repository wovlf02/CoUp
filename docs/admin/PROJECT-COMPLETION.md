## 💎 기술 하이라이트

### 아키텍처
```
Frontend: Next.js 15 + React 19
Backend: Next.js API Routes (서버리스)
Database: PostgreSQL + Prisma ORM
Auth: NextAuth.js 5 (JWT)
Styling: CSS Modules
Charts: Recharts
Deployment: Vercel / Docker
```

### 디자인 패턴
- **Server Components**: 기본 페이지
- **Client Components**: 인터랙션 컴포넌트
- **API Routes**: RESTful API
- **Middleware**: 인증 체크
- **Caching**: 메모리 캐시 (설정, 통계)

### 최적화
- **데이터베이스**: 인덱스 최적화, 쿼리 최적화
- **프론트엔드**: 코드 스플리팅, 동적 임포트
- **캐싱**: 메모리 캐시, HTTP 캐시
- **번들**: 트리 쉐이킹, 최소화
