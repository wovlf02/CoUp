## 🎨 디자인 시스템 적용

### 색상
```css
--admin-primary: #EF4444;      /* Red */
--admin-success: #10B981;      /* Green */
--admin-warning: #F59E0B;      /* Orange */
--admin-danger: #EF4444;       /* Red */
--admin-info: #3B82F6;         /* Blue */
```

### 타이포그래피
- 제목: 1.5rem, font-weight: 700
- 부제목: 1.125rem, font-weight: 600
- 본문: 0.875rem
- 작은 텍스트: 0.75rem

### 간격
- 섹션 간격: 24px
- 카드 간격: 16px
- 폼 그룹: 24px
- 입력 요소: 12px

### 반응형 브레이크포인트
- Desktop: 1920px+
- Tablet: 1024-1919px
- Mobile: <1024px
