# 개발 가이드

이 폴더에는 CoUp 프로젝트 개발 시 참고해야 할 가이드 문서들이 포함되어 있습니다.

## 📑 문서 목록

### 1. [navigation.md](./navigation.md)
- **네비게이션 구조 가이드**
- 앱 전체 네비게이션 구조
- 라우팅 규칙
- 메뉴 및 탭 구조

### 2. [feature-implementation.md](./feature-implementation.md)
- **기능 구현 가이드**
- 새로운 기능 추가 프로세스
- 코드 작성 규칙
- 테스트 및 검증 방법

### 3. [access-control.md](./access-control.md)
- **접근 제어 정책**
- 권한 관리 시스템
- 역할별 접근 권한 (OWNER, ADMIN, MEMBER)
- 페이지 및 API 레벨 권한 체크

## 🎯 사용 방법

### 새 기능 개발 시
1. **feature-implementation.md** 확인 → 구현 프로세스 이해
2. **access-control.md** 확인 → 권한 설정
3. **navigation.md** 참조 → 라우팅 추가

### 권한 관련 작업
- 페이지 접근 제어 설정
- API 권한 체크 추가
- 역할별 UI 표시/숨김
